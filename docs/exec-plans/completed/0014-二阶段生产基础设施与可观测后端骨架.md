## 2. 本轮目标

建立可运行、可迁移、可扩展的 FastAPI 生产后端基础，打通 PostgreSQL、Redis、Qdrant 真实依赖，完成每次系统执行的 Trace 数据模型、用户反馈入库和 bad case 自动归集。同时为 Agent Harness、Memory、Agentic RAG、Skill、Input/Output、Tool 模块建立 Protocol 边界，使后续实现可独立替换、测试和优化。

## 3. 范围

### 3.1 做什么

1. 建立 `apps/api` Python 3.12 / uv 工程，锁定 AgentScope 2.0.4，配置 ruff、mypy、pytest 和覆盖率门槛。
2. 建立 FastAPI 应用工厂、版本化 API、纯 ASGI 请求/Trace ID 与解析前 body limit 中间件、结构化脱敏日志、安全响应头和可配置 CORS。
3. 建立 PostgreSQL 16 异步连接池、Alembic 迁移与核心表：`users`、`sessions`、`messages`、`health_profiles`、`execution_traces`、`trace_events`、`user_feedback`、`bad_cases`。
4. 建立 Trace/Feedback 服务及 API：执行开始、事件追加、执行完成/失败、用户评价；负反馈或失败 Trace 自动产生 bad case。
5. Trace JSONB 只接收按事件类型 allowlist 的有限审计元数据；自由文本 PHI 列使用 AES-256-GCM envelope encryption，禁止 API Key、Authorization、Cookie、原始密码或 Chain-of-Thought 入库/日志。
6. 建立 Redis 和 Qdrant 异步适配器，实现 `/health/live`、`/health/ready`、`/metrics`；readiness 如实区分 PostgreSQL/Redis/Qdrant/AgentScope/知识库路径状态。
7. 按设计要求第四章为 Agent Harness、Memory、RAG、Skill、Input/Output、Tool 建立独立目录、README、测试、`Protocol` 和共享 Pydantic 契约，只定义可替换边界，不写虚假实现。
8. 建立 Dockerfile、生产基础 Compose、仅绑定 loopback 的开发 override、`.env.example` 和运维说明；数据网络默认 internal，本地知识库以只读卷挂载。
9. 使用真实 PostgreSQL/Redis/Qdrant 容器运行集成测试，不使用 mock 外部依赖声称成功。
10. 根据用户 2026-07-14 补充要求，以根目录 `.env` 为 canonical 配置源（Compose 不再加载旧 `apps/mvp/.env*`，不复制或提交密钥），补齐第四章环境契约；新增 opt-in 真实模型 smoke test。该项只验证连接与契约，不在本变更集提前实现聊天业务。
11. JWT scope/tenant/actor 由可信 token 派生，Repository 全链路显式 tenant 条件；受保护 API 使用 Redis 原子 100 req/min 限流，读事件分页，event/finish 完整幂等。

### 3.2 不做什么

- 本轮不实现 LLM 对话、处方生成、CGA、ASR/TTS、Skill 执行或 Agentic RAG 检索逻辑；这些属于后续独立变更集。
- 不用硬编码规则或模拟回复伪装医疗能力。
- 不在本轮替换 MVP 前端 API Client；前端迁移在后端对话契约完成后单独进行。
- 不在本轮声称“支撑万级并发”；只落实异步 I/O、连接池、无状态 API 和指标基础，最终能力必须经过专门压测证明。

## 4. 架构不变量

1. PostgreSQL 是长期业务数据与 Trace 的权威存储；Redis 仅承载会话热状态、限流、锁和 AgentScope message bus；Qdrant 只承载 RAG/长期记忆向量。
2. 应用层仅依赖 Protocol，具体数据库/外部 API 适配器通过 FastAPI dependency injection 注入。
3. 所有信任边界使用 Pydantic 运行时校验；前端契约后续使用同源 JSON Schema 生成 Zod 校验。
4. 不持久化模型私有 Chain-of-Thought；只存储可审计的决策摘要、工具调用、引用、延迟、Token 用量和安全判定。
5. 新增代码全量 Type Hints，公共类/方法有 docstring，无静默异常。

## 6. 风险与后续

- Python 3.14 为当前系统默认，AgentScope 及数据库依赖的生产组合将锁定 Python 3.12，由 uv 自动管理。
- AgentScope Agent Service 的 Redis 状态存储不替代 PostgreSQL 业务持久化；后续 Agent Harness 通过双写/事件方式接入 Trace 数据平面。
