## 1. 模块映射总览

GerClaw 的"医学工具集"模块承担 Agent 可调用医学能力的统一入口职责，覆盖三类工具供给渠道：

| 能力分类 | AgentScope 对应组件 | GerClaw 落地场景 |
|---------|-------------------|-----------------|
| **MCP 医疗工具服务** | `agentscope.mcp.MCPClient` + `StdioMCPConfig` / `HttpMCPConfig` → `MCPTool` | 用药查询、检验结果解读、医学计算器、文献检索、药物相互作用检查等可独立部署的医疗微服务 |
| **内置工具组合** | `agentscope.tool.Bash` / `Read` / `Write` / `Edit` / `Glob` / `Grep` | 沙箱内执行医学脚本（Python 生物统计、FHIR 数据解析）、读写患者病历摘要文件、批量处理检验报告 |
| **本地沙箱工作区** | `agentscope.workspace.LocalWorkspace` | 隔离医疗数据目录（`/data/gerclaw/sessions/<session_id>/`）、上下文/工具结果卸载、技能持久化 |
| **工具分组管理** | `agentscope.tool.ToolGroup` + `Toolkit` | 按场景分组：`basic`（默认工具）、`medication`（用药工具组）、`lab_result`（检验工具组）、`cga_assessment`（老年综合评估工具组） |
| **权限控制** | `agentscope.permission.PermissionEngine` + `PermissionContext` | Bash 写操作需 ASK、医疗数据目录外禁止 Write、只读检验查询默认 ALLOW |

**核心调用链路**：

```
Agent.reply()
  └─ Toolkit.call_tool(tool_call, state)
       ├─ MCPTool.call()  → MCPClient._session.call_tool(name, args) → MCP stdio/http 服务
       ├─ FunctionTool.call() → 本地 Python 函数（如医学计算器）
       └─ 内置工具(Bash/Read/Write) → Backend(LocalBackend/DockerBackend/E2B) → OS
```

**MCP 工具命名规则**：MCP 工具在 Agent 侧的统一名称为 `mcp__{mcp_name}__{sanitized_tool_name}`，非字母数字下划线连字符字符替换为 `x`。例如：

- MCP 名 `medical_tools` + 工具名 `get_drug_info` → `mcp__medical_tools__get_drug_info`

## 2. 核心 API 参考

### 2.1 MCPTool（MCP 工具适配器）

- **模块**: `agentscope.tool`
- **文件**: `src/agentscope/tool/_adapters.py`
- **基类**: `ToolBase`

MCPTool 是 AgentScope 对外部 MCP 工具的适配器包装，将 MCP 协议的 Tool 定义转换为 Agent 可调用的 ToolBase 实例。

```python
from agentscope.tool import MCPTool

# 通常不直接构造，而是通过 MCPClient.get_tool() / list_tools() 获取
```

| 属性 | 值 | 说明 |
|------|-----|------|
| `is_mcp` | `True` | MCP 工具标记 |
| `is_state_injected` | `False` | 禁止状态注入（安全） |
| `mcp_name` | str | 所属 MCP 服务器名 |

**权限策略**：只读 MCP 工具自动返回 `ALLOW`，写类工具返回 `ASK`。

### 2.2 StdioMCPConfig / HttpMCPConfig（MCP 连接配置）

- **模块**: `agentscope.mcp`
- **文件**: `src/agentscope/mcp/_config.py`

**StdioMCPConfig（标准输入输出模式，本地进程）**：

```python
from agentscope.mcp import StdioMCPConfig

config = StdioMCPConfig(
    command="python",                         # 启动命令
    args=["-m", "gerclaw_medical_mcp"],       # 命令行参数
    env={"PYTHONPATH": "/opt/gerclaw"},       # 环境变量（可选）
    cwd="/opt/gerclaw/services",              # 工作目录（可选）
    encoding_error_handler="strict",          # 编码错误处理
)
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | `"stdio_mcp"` | 默认 | 判别器字段 |
| `command` | str | 是 | 启动 MCP 服务器的命令 |
| `args` | list[str] | 否 | 命令行参数 |
| `env` | dict[str,str] | 否 | 环境变量 |
| `cwd` | str/Path | 否 | 工作目录 |

**HttpMCPConfig（HTTP/SSE 模式，远程服务）**：

```python
from agentscope.mcp import HttpMCPConfig

config = HttpMCPConfig(
    url="http://medical-mcp.gerclaw.local:8080/sse",
    headers={"Authorization": "Bearer xxx"},   # 可选
    timeout=30.0,                               # 默认30秒
)
```

### 2.3 MCPClient（统一 MCP 客户端）

- **模块**: `agentscope.mcp`
- **文件**: `src/agentscope/mcp/_mcp_client.py`
- **基类**: `pydantic.BaseModel`

```python
from agentscope.mcp import MCPClient, StdioMCPConfig

# 1. 创建客户端（STDIO 必须 is_stateful=True）
client = MCPClient(
    name="medical_tools",            # 必须匹配 [a-zA-Z0-9_-]+
    is_stateful=True,                # STDIO 必须 True
    mcp_config=StdioMCPConfig(
        command="python",
        args=["mcp_medical_server.py"],
    ),
    enable_tools=None,               # 白名单，None=全部
    disable_tools=None,              # 黑名单
    execution_timeout=30.0,          # 工具执行超时
)

# 2. 建立连接（有状态模式必须显式 connect）
await client.connect()

# 3. 列出所有工具 → 返回 list[MCPTool]
tools = await client.list_tools()

# 4. 获取单个工具
drug_tool = await client.get_tool("get_drug_info")

# 5. 调用工具
result = await drug_tool(drug_name="二甲双胍")
print(result.content[0].text)

# 6. 关闭连接
await client.close()
```

| 方法 | 说明 |
|------|------|
| `connect()` | 建立有状态连接（STDIO/HTTP 有状态模式必须调用） |
| `close()` | 关闭连接并释放子进程/HTTP 资源 |
| `list_tools()` | 返回该 MCP 服务暴露的所有 `MCPTool` 列表 |
| `get_tool(name)` | 按原始名称获取单个工具（绕过 enable/disable 过滤） |
| `is_connected` | 属性，返回当前连接状态 |

**关键验证规则**：
- `name` 必须匹配正则 `[a-zA-Z0-9_-]+`（LLM 工具名限制）
- STDIO 类型必须 `is_stateful=True`
- `enable_tools` 和 `disable_tools` 不可同时指定且不可重叠

### 2.4 内置工具 Bash / Read / Write 组合用法

```python
from agentscope.tool import Bash, Read, Write, Toolkit

# 内置工具共享 Backend 抽象层，默认 LocalBackend
bash = Bash(cwd="/data/gerclaw/workspace")    # 执行命令
read = Read(max_line_characters=2000)          # 读文件
write = Write()                                # 写文件

# Bash 参数: command (str)
# Read 参数: file_path (str), offset (int,可选), limit (int,可选)
# Write 参数: file_path (str), content (str)

toolkit = Toolkit(tools=[bash, read, write])
```

**Bash 只读命令自动放行**：通过 tree-sitter-bash 解析器判定，`ls`、`cat`、`git status` 等只读命令自动通过权限检查；写命令（`rm`、`mkdir`、重定向）默认 ASK。

**典型医疗组合场景**：
- `Bash` 执行 Python 医学统计脚本：`python analyze_lab.py --input report.csv`
- `Read` 读取检验报告 JSON：读取 `/data/gerclaw/sessions/xxx/lab_result.json`
- `Write` 输出 AI 辅助诊断摘要：写入 `diagnosis_summary.md`

### 2.5 LocalWorkspace（本地沙箱工作区）

- **模块**: `agentscope.workspace`
- **文件**: `src/agentscope/workspace/_local_workspace.py`

```python
from agentscope.workspace import LocalWorkspace
from agentscope.mcp import MCPClient, StdioMCPConfig

workspace = LocalWorkspace(
    workdir="/data/gerclaw/workspace",         # 沙箱根目录（必填）
    default_mcps=[                              # 默认MCP服务（持久化到 .mcp 文件）
        MCPClient(
            name="medical_tools",
            is_stateful=True,
            mcp_config=StdioMCPConfig(
                command="python",
                args=["/opt/gerclaw/mcp_medical_server.py"],
            ),
        ),
    ],
    skill_paths=["/opt/gerclaw/skills/geriatric"],  # 技能目录，初始化时复制
    instructions="你有一个本地工作区在 {workdir}...",
)

await workspace.initialize()  # 创建目录、加载MCP、复制技能
```

**目录结构**（initialize 后）：
```
{workdir}/
├── sessions/<session_id>/
│   ├── context.jsonl        # 上下文卸载（JSONL）
│   └── tool_result-<id>.txt # 长工具结果卸载
├── data/<hash>.<ext>        # DataBlock 二进制持久化
├── skills/                  # 复制过来的技能
│   └── .skills              # 技能索引
└── .mcp                     # 持久化的 MCP 配置
```

| 方法 | 说明 |
|------|------|
| `initialize()` | 初始化工作区（创建目录、启动默认MCP、复制技能） |
| `offload_context(session_id, msgs)` | 将消息卸载到 JSONL 文件 |
| `offload_tool_result(session_id, result)` | 将长工具结果写入文件 |
| `list_mcps()` | 列出当前已连接的 MCP |
| `list_skills()` | 列出已加载技能 |

### 2.6 ToolGroup（工具分组管理）

- **模块**: `agentscope.tool`
- **文件**: `src/agentscope/tool/_tool_group.py`

```python
from agentscope.tool import ToolGroup, Toolkit, Bash, Read
from agentscope.mcp import MCPClient, StdioMCPConfig

# 用药工具组
medication_group = ToolGroup(
    name="medication",
    description="老年患者用药相关工具：药物查询、相互作用检查、剂量计算",
    instructions="当患者咨询用药问题时激活此工具组",
    mcps=[drug_mcp_client],
)

# 检验工具组
lab_group = ToolGroup(
    name="lab_result",
    description="检验报告解读工具组",
    tools=[Read()],  # 内置读工具
)

# basic 组始终存在
toolkit = Toolkit(
    tools=[Bash()],                          # basic 组工具
    tool_groups=[medication_group, lab_group],  # 额外分组
)
```

Agent 通过 `ResetTools` 元工具动态激活/停用工具组。

## 6. GerClaw 适配要点

### 6.1 医学 MCP 服务设计

GerClaw 推荐将医学能力封装为独立的 MCP stdio 服务（`mcp.server.fastmcp.FastMCP`），核心工具包括：

| 工具名 | 功能 | 输入 | 输出 |
|--------|------|------|------|
| `get_drug_info` | 用药查询（适应症、用法用量、禁忌症、老年注意事项） | `drug_name: str` | 结构化药物信息文本 |
| `check_drug_interaction` | 药物相互作用检查 | `drug_list: list[str]` | 相互作用等级和说明 |
| `calculate_bmi` | BMI 计算 | `height_cm: float, weight_kg: float` | BMI 值 + 老年营养评估分级 |
| `calculate_creatinine_clearance` | 肌酐清除率（Cockcroft-Gault，老年肾功能评估核心） | `age: int, weight_kg: float, scr_mg_dl: float, sex: str` | Ccr 值 + 剂量调整建议 |
| `interpret_lab_result` | 检验结果解读 | `test_name: str, value: float, unit: str, ref_range: str` | 异常判定 + 临床意义 |
| `search_guidelines` | 老年医学指南检索 | `query: str` | 指南推荐意见摘要 |
| `cga_score` | CGA 老年综合评估打分 | `domain: str, answers: dict` | 评估得分 + 干预建议 |

**服务部署建议**：
- 使用 `FastMCP("gerclaw_medical")` 注册工具，通过 `server.run(transport="stdio")` 启动
- 对外部医学数据库（如药品说明书库、临床指南库）的访问封装在 MCP 服务内，Agent 只通过标准 MCP 协议访问
- 工具描述（docstring）要面向老年医学场景，明确标注"老年患者注意事项"

### 6.2 推荐配置：LocalWorkspace 隔离医疗数据目录

```python
# GerClaw 推荐的工作区配置
workspace = LocalWorkspace(
    workdir="/data/gerclaw/sessions",        # 每个会话独立子目录
    default_mcps=[medical_mcp_client],       # 默认加载医学 MCP
    skill_paths=["/opt/gerclaw/skills/"],
    instructions=(
        "你是 GerClaw 老年医疗助手，工作目录 {workdir}。\n"
        "患者隐私数据仅可写入该目录，禁止访问其他路径。"
    ),
)
```

**隔离要点**：
- 每个会话（session_id）自动创建独立子目录，避免患者数据交叉污染
- DataBlock（如检验报告 PDF、医学影像）自动哈希去重持久化到 `data/`
- 长对话自动触发上下文卸载，防止医疗问诊历史超出上下文窗口
- `.mcp` 配置持久化，服务重启后自动重连医学 MCP 服务

### 6.3 风险：Bash 工具需严格权限控制

Bash 工具在医疗场景风险极高，必须严格控制：

| 风险场景 | 控制措施 |
|---------|---------|
| 执行任意系统命令 | 使用 `PermissionMode.EXPLORE`（只读模式）或自定义 deny 规则 |
| 删除/修改患者数据 | `dangerous_directories` 加入医疗数据根目录；deny 规则匹配 `rm`、`mv`、`>` 等写命令 |
| 泄露隐私数据到外网 | deny 规则匹配 `curl`、`wget`、`nc` 等网络命令；或使用 DockerBackend 网络隔离 |
| 注入恶意命令 | tree-sitter 解析已有基础防护，但医疗场景建议使用白名单命令 |
| 长时间阻塞进程 | 设置 `execution_timeout`（建议医疗工具 ≤30s） |

**推荐权限配置**：
```python
from agentscope.permission import (
    PermissionContext, PermissionMode, PermissionRule, PermissionBehavior
)

# 医疗场景推荐 EXPLORE 模式：只读默认，写操作显式授权
context = PermissionContext(mode=PermissionMode.EXPLORE)
# 添加显式 ALLOW 规则（如允许 Python 脚本执行）
context.allow_rules["Bash"] = [
    PermissionRule(
        tool_name="Bash",
        rule_content="python /opt/gerclaw/scripts/",
        behavior=PermissionBehavior.ALLOW,
        source="gerclaw_config",
    ),
]
```

## 7. 可运行示例指引

本参考附带两个可运行示例，位于 `agentscope-examples/17_mcp_tools/`：

| 文件 | 功能 | 运行前置条件 |
|------|------|------------|
| `mcp_medical_server.py` | MCP stdio 医疗工具服务端，提供 `get_drug_info`（用药查询）和 `calculate_bmi`（BMI 计算）两个工具 | `pip install mcp fastmcp`（AgentScope 已依赖 mcp） |
| `mcp_client_integration.py` | 通过 MCPClient + StdioMCPConfig 连接上述服务，注册到 Agent Toolkit，模拟张大爷咨询用药场景 | 需要配置 `DASHSCOPE_API_KEY` 环境变量；`pip install agentscope dashscope` |

**运行步骤**：

```bash
# 1. 安装依赖（AgentScope 环境已包含 mcp）
pip install agentscope "mcp[cli]" dashscope

# 2. 配置 API Key
export DASHSCOPE_API_KEY="sk-your-key-here"

# 3. 进入示例目录
cd agentscope-examples/17_mcp_tools

# 4. 运行客户端集成示例（自动启动 stdio MCP 子进程）
python mcp_client_integration.py
```

**示例场景**：模拟张大爷（72岁，身高170cm，体重78kg，患有2型糖尿病）咨询二甲双胍用药。Agent 会自动调用：
1. `calculate_bmi` 计算张大爷 BMI，判断超重
2. `get_drug_info` 查询二甲双胍的适应症、用法、老年注意事项
3. 综合给出面向老年人的用药建议

**扩展练习**：
- 在 `mcp_medical_server.py` 中添加 `check_drug_interaction` 工具
- 配置 `LocalWorkspace` 将对话历史持久化到本地目录
- 使用 `ToolGroup` 将用药工具与评估工具分组管理
- 添加 `PermissionContext` 限制 Bash 只能运行指定医学脚本
