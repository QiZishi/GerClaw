# MMSE 简易智能精神状态检查量表

## Mini-Mental State Examination

### 一、量表简介

MMSE（Mini-Mental State Examination）由 Folstein 等人于 1975 年编制，是国内外广泛应用的痴呆筛选首选量表。该量表简单易行，评估内容包括定向力、记忆力、注意力和计算力、回忆能力、语言能力等方面。

**评估时间范围**：即时评估

**适用人群**：成人（特别适用于老年人）

**评估时长**：约5-10分钟

---

### 三、评分标准

#### 3.1 计分方法
- 每1项正确为1分，错误为0分
- 总分范围为0～30分

#### 3.2 结果判定

| 总分范围 | 认知功能状态 | 临床建议 |
|:---:|---------|---------|
| 27-30分 | 认知功能正常 | 定期随访 |
| 21-26分 | 轻度认知功能障碍 | 建议进一步评估 |
| 10-20分 | 中度认知功能障碍 | 建议精神科就诊 |
| 0-9分 | 重度认知功能障碍 | 建议立即精神科就诊 |

#### 3.3 教育程度校正

**分界值与教育程度有关**：

| 教育程度 | 分界值 | 说明 |
|---------|:---:|------|
| 文盲（未受教育） | ≤17分 | 认知功能障碍 |
| 小学（受教育年限≤6年） | ≤20分 | 认知功能障碍 |
| 中学或以上（受教育年限＞6年） | ≤24分 | 认知功能障碍 |

**分界值以下为有认知功能缺陷，以上为正常。**

---

### 四、特殊条目说明

1. **重复**（第24条）：必须完全相同才算正确
2. **阅读**（第25条）：有闭眼睛的动作才给分
3. **绘图**（第30条）：图要有10个角和2条相交的直线

---

### 五、使用注意事项

1. **评估环境**：确保患者在安静、光线充足的环境中完成评估
2. **独立完成**：患者应独立完成，避免他人提示或帮助
3. **记录详细**：详细记录患者的回答
4. **文化适应**：需考虑文化背景和教育程度的影响
5. **非诊断工具**：MMSE是筛查工具，不能替代临床诊断
