# SAS 焦虑自评量表

## Self-Rating Anxiety Scale

### 一、量表简介

SAS（Self-Rating Anxiety Scale）由 Zung 于 1971 年编制，是用于测量焦虑状态轻重程度及其在治疗过程中变化情况的心理量表。该量表含有20个反映焦虑主观感受的项目，适用于心理咨询门诊、精神科门诊或住院精神病人。

**评估时间范围**：过去一周

**适用人群**：成人（18岁以上）

**评估时长**：约5-10分钟

---

### 二、量表内容

**指导语**：下面有20条文字，请仔细阅读每一条，把意思弄明白，然后根据您最近一周的实际感觉，在分数栏1～4分中选择与你的情况相符的"√"。

| 序号 | 条目内容 | 没有或很少 (1分) | 有时有 (2分) | 大部分时间 (3分) | 绝大部分时间 (4分) |
|:---:|---------|:---:|:---:|:---:|:---:|
| 1 | 我觉得比平常容易紧张或着急 | ☐ | ☐ | ☐ | ☐ |
| 2 | 我无缘无故地感到害怕 | ☐ | ☐ | ☐ | ☐ |
| 3 | 我容易心里烦乱或觉得惊恐 | ☐ | ☐ | ☐ | ☐ |
| 4 | 我觉得我可能将要发疯 | ☐ | ☐ | ☐ | ☐ |
| 5* | 我觉得一切都很好，也不会发生什么不幸 | ☐ | ☐ | ☐ | ☐ |
| 6 | 我手脚发抖打颤 | ☐ | ☐ | ☐ | ☐ |
| 7 | 我因为头痛、颈痛和背痛而苦恼 | ☐ | ☐ | ☐ | ☐ |
| 8 | 我感觉容易衰弱和疲乏 | ☐ | ☐ | ☐ | ☐ |
| 9* | 我觉得心平气和，并且容易安静坐着 | ☐ | ☐ | ☐ | ☐ |
| 10 | 我觉得心跳得很快 | ☐ | ☐ | ☐ | ☐ |
| 11 | 我因为阵阵头晕而苦恼 | ☐ | ☐ | ☐ | ☐ |
| 12 | 我有晕倒发作，或觉得要晕倒似的 | ☐ | ☐ | ☐ | ☐ |
| 13* | 我吸气呼气都感到很容易 | ☐ | ☐ | ☐ | ☐ |
| 14 | 我的手脚麻木和刺痛 | ☐ | ☐ | ☐ | ☐ |
| 15 | 我因为胃痛和消化不良而苦恼 | ☐ | ☐ | ☐ | ☐ |
| 16 | 我常常要小便 | ☐ | ☐ | ☐ | ☐ |
| 17* | 我的手脚常常是干燥温暖的 | ☐ | ☐ | ☐ | ☐ |
| 18 | 我脸红发热 | ☐ | ☐ | ☐ | ☐ |
| 19* | 我容易入睡并且一夜睡得很好 | ☐ | ☐ | ☐ | ☐ |
| 20 | 我做恶梦 | ☐ | ☐ | ☐ | ☐ |

> **注**：带*号的条目为反向计分题（第5、9、13、17、19条），计分时需反向计算（4→1，3→2，2→3，1→4）。

---

### 三、评分标准

#### 3.1 计分方法
1. 将20个条目的得分相加，得到粗分（Raw Score）
2. 将粗分乘以1.25，四舍五入取整数，得到标准分（Standard Score）

#### 3.2 结果判定

| 标准分范围 | 焦虑程度 | 临床建议 |
|:---:|---------|---------|
| <50分 | 无焦虑 | 正常 |
| 50-59分 | 轻度焦虑 | 建议心理咨询 |
| 60-69分 | 中度焦虑 | 建议精神科就诊 |
| ≥70分 | 重度焦虑 | 建议立即精神科就诊 |

---

### 四、临床分级说明

关于焦虑症状的临床分级，除参考量表分值外，主要还应根据临床症状，特别是**要害症状**的程度来划分：

1. **与处境不相称的痛苦情绪体验**：过度担忧、恐惧
2. **精神运动性不安**：坐立不安、来回走动、搓手顿足
3. **植物神经功能障碍**：心悸、出汗、口干、尿频等

---

### 五、使用注意事项

1. **评估时间**：请根据最近一周的实际感觉进行评估
2. **独立完成**：患者应独立完成，避免他人干扰
3. **反向计分**：注意第5、9、13、17、19条为反向计分题
4. **临床结合**：量表结果仅供参考，需结合临床症状综合判断
5. **非诊断工具**：SAS是筛查工具，不能替代临床诊断

### 七、附录：GAD-7焦虑量表（对比参考）

GAD-7是另一种常用的焦虑筛查工具，与SAS相比：
- GAD-7更侧重于广泛性焦虑
- GAD-7评估时间更短（约2分钟）
- GAD-7在初级医疗中更常用

**GAD-7评分标准**：
- 0-4分：无焦虑
- 5-9分：轻度焦虑
- 10-14分：中度焦虑
- 15-21分：重度焦虑
