# 精神心理评估问卷量表集合

本文件夹包含五个常用的精神心理评估问卷量表，用于评估抑郁、焦虑、睡眠质量和认知功能。

## 量表用途

### 1. PHQ-9 快速抑郁评估量表
- **用途**：筛查抑郁症状，评估抑郁严重程度
- **适用人群**：成人（18岁以上）
- **评估时间范围**：过去两周
- **评分标准**：0-27分，分为无抑郁、轻度、中度、中重度、重度五个等级

### 2. SAS 焦虑自评量表
- **用途**：评估焦虑症状的轻重程度
- **适用人群**：成人（18岁以上）
- **评估时间范围**：过去一周
- **评分标准**：标准分<50分为无焦虑，50-59分为轻度焦虑，60-69分为中度焦虑，≥70分为重度焦虑

### 3. PSQI 匹兹堡睡眠质量指数量表
- **用途**：评估睡眠质量
- **适用人群**：成人（18岁以上）
- **评估时间范围**：过去一个月
- **评分标准**：0-21分，分为睡眠质量很好、较好、一般、差四个等级

### 4. Mini-Cog 简易认知量表
- **用途**：筛查轻度认知障碍和早期痴呆
- **适用人群**：老年人（65岁以上）
- **评估时长**：约3分钟
- **评分标准**：0-5分，0-2分可能存在认知障碍，3-5分认知功能正常

### 5. MMSE 简易智能精神状态检查量表
- **用途**：痴呆筛选的首选量表
- **适用人群**：成人（特别适用于老年人）
- **评估时长**：约5-10分钟
- **评分标准**：0-30分，根据教育程度有不同的分界值

## 使用建议

1. **评估环境**：确保患者在安静、私密的环境中完成评估
2. **独立完成**：患者应独立完成，避免他人干扰
3. **评估频率**：建议每2-4周重复评估，以监测症状变化
4. **文化适应**：需考虑文化背景对症状表达的影响
5. **非诊断工具**：这些量表都是筛查工具，不能替代临床诊断
