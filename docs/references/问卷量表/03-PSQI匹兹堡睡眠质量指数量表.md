# PSQI 匹兹堡睡眠质量指数量表

## Pittsburgh Sleep Quality Index

### 一、量表简介

PSQI（Pittsburgh Sleep Quality Index）由美国匹兹堡大学精神科医师 Buysse 博士等于 1993 年编制，主要用于评定被试者近1个月的睡眠质量。该量表由19个自我评定问题和5个由睡眠同伴评定问题组成，但仅需对19个自我评定问题进行计分。

**评估时间范围**：过去一个月

**适用人群**：成人（18岁以上）

**评估时长**：约5-10分钟

---

### 二、量表内容

**指导语**：下面一些问题是关于您最近一个月的睡眠状况，这仅仅与您的睡眠习惯有关。请选择或填写最符合您近一个月白天和晚上实际情况的选项，并尽可能地做精确回答。

#### 自我评定部分（共19个条目）

| 序号 | 项目内容 | 选项/评分标准 |
|:---:|---------|------------|
| 1 | 过去1个月，你通常上床睡觉的时间是？ | _____点_____分 |
| 2 | 过去1个月，你每晚通常要多长时间才能入睡？ | 0分：≤15分钟<br>1分：16~30分钟<br>2分：31~60分钟<br>3分：>60分钟 |
| 3 | 过去1个月，你每天早上通常什么时候起床？ | _____点_____分 |
| 4 | 过去1个月，你每晚实际睡眠的时间有多少？ | _____小时_____分钟 |
| 5A | 不能在30分钟内入睡 | 0分：过去1个月没有<br>1分：每周平均不足1个晚上<br>2分：每周平均1~2个晚上<br>3分：每周平均3个或更多晚上 |
| 5B | 在晚上睡眠中醒来或早醒 | 同上 |
| 5C | 晚上有无起床上洗手间 | 同上 |
| 5D | 不舒服的呼吸 | 同上 |
| 5E | 大声咳嗽或打鼾 | 同上 |
| 5F | 感到寒冷 | 同上 |
| 5G | 感到太热 | 同上 |
| 5H | 做噩梦 | 同上 |
| 5I | 出现疼痛 | 同上 |
| 5J | 其他影响睡眠的事情（如有请说明） | 同上 |
| 6 | 你对过去1个月睡眠质量总的评价 | 0分：非常好<br>1分：尚好<br>2分：不好<br>3分：非常差 |
| 7 | 近1个月你用催眠药物的情况 | 同5A评分标准 |
| 8 | 过去1个月你在开车、吃饭或参加社会活动时难以保持清醒状态？ | 同5A评分标准 |
| 9 | 过去1个月你在积极完成事情上有无困难？ | 0分：没有困难<br>1分：有一点困难<br>2分：比较困难<br>3分：非常困难 |
| 10 | 你是与人同睡一床（睡觉同伴，包括配偶等）或有室友？ | 选项1：没有与人同睡一床或有室友<br>选项2：同伴或室友在另外房间<br>选项3：同伴在同一房间但不睡同床<br>选项4：同伴在同一床上 |

#### 同伴评定部分（共5个条目）

如果你是与人同睡一床或有室友，请询问他（她）在过去1个月是否出现以下情况：

| 序号 | 项目内容 | 选项 |
|:---:|---------|------|
| 11 | 你在睡觉时，有无大鼾声 | 选项1：过去1个月没有<br>选项2：每周平均不足1个晚上<br>选项3：每周平均1~2个晚上<br>选项4：每周平均3个或更多晚上 |
| 12 | 你在睡觉时，呼吸之间有没有长时间停顿 | 同上 |
| 13 | 你在睡觉时，你的腿有无抽动或者有痉挛 | 同上 |
| 14 | 你在睡觉时是否出现不能辨认方向或混乱状态 | 同上 |
| 15 | 你在睡觉时你有无其他睡不安宁的情况（若有，请描述） | 同上 |

### 三、评分标准

#### 3.1 各维度（因子）计分方法

PSQI由19个自评条目构成7个维度（因子），每个因子分0-3分，总分0-21分。

| 因子 | 计分方法 | 得分 |
|-----|---------|:---:|
| **A. 睡眠质量** | 根据条目6的应答计分 | |
| **B. 入睡时间** | 累加条目2和5A的计分：<br>"0"计0分<br>"1~2"计1分<br>"3~4"计2分<br>"5~6"计3分 | |
| **C. 睡眠时间** | 根据条目4的应答计分：<br>">7h"计0分<br>"6~7h（不含6h）"计1分<br>"5~6h（含6h）"计2分<br>"<5h"计3分 | |
| **D. 睡眠效率** | 睡眠效率=条目4（睡眠时间）/床上时间×100%<br>注：床上时间=条目3（起床时间）-条目1（上床时间）<br>睡眠效率>85%计0分<br>75%~84%计1分<br>65%~74%计2分<br><65%计3分 | |
| **E. 睡眠障碍** | 累加条目5B~5J的计分：<br>"0"则计0分<br>"1~9"计1分<br>"10~18"计2分<br>"19~27"计3分 | |
| **F. 催眠药物** | 根据条目7的应答计分 | |
| **G. 日间功能障碍** | 累加条目8和9的得分：<br>"0"则计0分<br>"1~2"计1分<br>"3~4"计2分<br>"5~6"计3分 | |

**PSQI总分 = 因子A + 因子B + 因子C + 因子D + 因子E + 因子F + 因子G**

#### 3.2 结果判定

| 总分范围 | 睡眠质量 | 临床建议 |
|:---:|---------|---------|
| 0~5分 | 睡眠质量很好 | 维持良好睡眠习惯 |
| 6~10分 | 睡眠质量较好 | 注意睡眠卫生 |
| 11~15分 | 睡眠质量一般 | 建议改善睡眠习惯 |
| 16~21分 | 睡眠质量差 | 建议就医咨询 |

---

### 四、使用注意事项

1. **评估时间**：请根据最近一个月的实际睡眠情况进行评估
2. **独立完成**：患者应独立完成，避免他人干扰
3. **精确填写**：尽量精确填写睡眠时间，避免估算
4. **同伴评定**：如有室友或配偶，建议请其协助完成第11-15条
5. **文化适应**：需考虑文化背景对睡眠习惯的影响

### 六、附录：睡眠效率计算示例

**计算公式**：
```
睡眠效率 = (实际睡眠时间 / 床上时间) × 100%
床上时间 = 起床时间 - 上床时间
```

**示例**：
- 上床时间：23:00
- 起床时间：7:00
- 实际睡眠时间：6小时

计算：
- 床上时间 = 7:00 - 23:00 = 8小时
- 睡眠效率 = (6/8) × 100% = 75%
- 对应因子D得分：1分
