# 肌少症膳食指导与营养补充剂使用共识

何书励 刘鹏举 王勃诗 王方 马方

仲图分类号 R592; R151 交献标志码 A

肌肉衰减症又称“肌少症”，是由Rosenberg于1989年首次命名。2010年，欧洲老年人肌少症工作组（European Working Group on Sarcopenia in Older People，EWGSOP）发表了肌少症共识，此后，不同国家和组织对肌少症诊断标准进行了修改。目前，肌少症已经被正式命名为一种肌肉衰竭性疾病，常见于老年人，也可发生于年轻人。肌肉力量降低是肌少症的关键特征，肌肉数量和质量降低是其诊断依据，躯体功能不佳是严重肌少症的标志。为提高老年肌少症膳食指导和营养补充剂使用的科学性和规范性，本项目组制订了本共识，适合临床营养科和老年科、养老机构等相关工作人员使用。

# 第一部分：膳食指导共识

# 1推荐：建议采用地中海饮食等健康膳食模式，首选自然食物，重视食物规划

《营养在社区老年人肌肉健康中的作用——亚洲肌肉减少症工作组循证专家共识》指出，在可行的情况下，医疗保健专业人员应在膳食强化或补充之前为存在营养不良或肌少症风险的人提供膳食建议和营养咨询，蛋白质应优先通过饮食获取，如不能则考虑蛋白质补充。地中海和日本膳食模式可降低肌少症风险，而西方膳食模式则增加风险；在防治肌少症方面，经过良好规划的饮食可以取得与营养补充一样（或许更好）的改善肌少症的效果。

地中海饮食是健康膳食模式的代表，具有以下特点：蔬菜、水果、全谷类、豆类和坚果摄入量较高；适量摄入奶制品；适量摄入红酒和鱼类等海产品；肉类及其

制品摄入量较低；食物加工程度低而新鲜度高；橄榄油是主要的食用油，也是主要的脂肪来源。以色列一项横断面研究显示，对地中海饮食依从性较高的女性具有更高的肌肉量。芬兰的一项研究也显示，地中海饮食评分更高的女性，其步速更快，下肢肌肉力量更强，并能更好地完成深蹲。日本饮食的特点是常用大米、味噜汤、鱼类和贝类、绿色和黄色蔬菜、海藻、腌菜等食物，而较少食用牛肉、猪肉和咖啡，日本饮食指数评分高与更低的肌少症患病率相关。

# 2 食物选择推荐

推荐：食物多样，谷类为主，杂粮占  $1 / 4\sim 1 / 2$

谷类中富含的碳水化合物是神经系统的主要能源，杂粮富含B族维生素（维生素B1、烟酸、维生素B2）、镁等，这些营养素在维持神经肌肉的线粒体功能、抑制凋亡中发挥着重要作用。

推荐：适量吃鱼、禽、蛋、瘦肉、奶制品、豆制品，保证优质蛋白摄入。

鱼肉富含  $\mathrm{n - 3}$  脂肪酸和维生素D；动物性蛋白质中支链氨基酸含量较高，鱼肉和鸡肉的支链氨基酸含量尤为丰富。牛奶中乳清蛋白的吸收较快，并且含有丰富的支链氨基酸，尤其是亮氨酸。乳清蛋白属于快消化蛋白，可引起餐后更高、更快的血液必需氨基酸浓度的上升，从而更好地促进肌肉蛋白质合成，提高肌肉量和肌肉功能。动物蛋白和植物蛋白联合可以进一步提高餐后肌肉蛋白质合成反应。

推荐：多吃深色蔬菜水果

英国一项随机对照试验显示，与每天摄入2份蔬菜水果相比，每天摄入5份蔬菜具有增加握力的趋势，蔬菜水果可能通过抗氧化作用（如类胡萝卜素）和碱盐发挥保存肌肉量的作用。

推荐：控烟限酒

如饮酒，一天摄入的酒精不超过  $15\mathrm{g}^{[14]}$  ；建议吸烟者积极戒烟，戒烟后食欲增加时，注意多增加蔬菜、

水果等食物的摄入，不要吃高热量的零食[4]。

吸烟是引起COPD的主要原因，COPD可以增加肌少症的风险，其机制为一方面烟碱通过升高瘦素（leptin）、降低饥饿素、损伤口腔健康等机制降低食欲，导致食物摄入减少，另一方面，COPD使人体处于炎症状态，进而导致肌肉量下降和功能丧失。饮酒则通过增加自噬、破坏肠道微生态等机制诱发肌少症[4]。

特殊食品：蜂产品

动物研究发现，蜂王浆可改善实验鼠的肌力和躯体功能，减少骨骼肌脂肪积聚，减缓肌肉减少，增加肌卫星细胞的分化和增殖，改善肌肉的再生，其机制可能包括促进肌肉合成、抑制分解代谢基因表达、拮抗代谢异常、抗炎、抗氧化应激、促进周围神经再生、改善肌肉血供、改善肠道微生态等[4]。然而，由于尚缺乏人体试验研究数据，暂不作推荐；未来需要对蜂产品在肌少症防治中的作用开展进一步的研究。

# 3 特殊人群推荐

肥胖：建议对合并肥胖的肌少症病人进行适度能量限制，达到既防治肌少症又不会引起营养不良的目的。

少肌型肥胖已成为重要的健康挑战。一项在老年衰弱人群中进行的随机对照试验显示，联合运动和饮食控制（较能量需要量减少  $500\sim 750\mathrm{kcal}$  比单独运动或饮食控制更能提高力量、平衡、步速、峰值氧耗、功能状态问卷评分，并且肌肉和骨密度下降更少[4]。适度能量限制延缓肌少症的机制包括减少氧自由基，提高线粒体功能，改善蛋白质平衡，减少铁过载，增加自噬，抑制凋亡，减少炎症等，然而过度的能量限制可能会促进肌肉量丢失并提高残疾和死亡率[4]。

肝硬化推荐：充足能量、蛋白质、肾意适量选择芳香族氨基酸含量较低的植物蛋白，常用支链氨基酸丰富的食物（鱼肉、鸡肉、牛奶），重视夜间加餐。

肝硬化病人存在以下恶性循环：一方面，肌肉的减少使机体代谢氨的能力下降；另一方面，高氨血症可通过肌肉抑制素（myostatin）、线粒体功能障碍、细胞应激反应进一步加剧肌肉消耗。可根据病人的体型、营养状况给予适当的能量（  $25\sim 40\mathrm{kcal / kg}$  ）、蛋白质（  $1.2\sim$ $1.5\mathrm{g / kg}$  ），肝性脑病病人需要适当减少蛋白质的摄入[4]。有研究显示，对于肝硬化病人，深夜加餐（21： $00\sim 23:00)$  至少  $50\mathrm{g}$  碳水化合物有助于增加瘦体组织[4]，支链氨基酸补充可提高肝硬化病人肌肉力量[4]。

肾病推荐：根据病人的肾功能、疾病活动、依从性等因素制订个体化营养方案，并与运动相配合，达到延缓或逆转肌少症的目的。注意保证透析病人充足的蛋白质摄入量。

慢性肾脏病病人在晚期需要限制蛋白质摄入量，因此罹患肌少症的风险增加。慢性肾脏病和肌少症的治疗在蛋白质摄入量方面存在矛盾，目前尚不能确定慢性肾脏疾病合并肌少症病人的蛋白质摄入量，建议根据病人的肾功能、疾病活动、依从性等因素，个体化制定合理的蛋白质摄入量。单纯的饮食干预常不能有效改善骨骼肌肌量、躯体功能和肌力，因此饮食必需与运动相结合，病人的运动方案应是个体化的，并需要在监护下进行[4]。

吞咽障碍推荐：需要对吞咽困难的病人进行评估，如能经口进食，提供不同咀嚼难度的食物。

肌少症（特别是咀嚼肌的减少）和吞咽障碍常互为因果，导致病人进食量不能满足身体需要量，形成恶性循环。可根据吞咽障碍评估情况，选择不同稠度的液体食物和不同质地的固体食物。如不能经口进食或经口进食量不足目标量的  $60\%$  ，应给予管饲喂养[4]。

# 第二部分：膳食补充剂使用共识

1 蛋白质口服营养补充剂：当膳食中蛋白质和（或）能量摄入不足时，可应用蛋白质和（或）含蛋白质营养制剂的口服补充；蛋白质补充量不少于  $20\mathrm{g / d}$

肌少症是一种受多因素影响的增龄性疾病，充足的能量和蛋白质摄入是相关危险因素中的可改变因素之一。膳食中蛋白质摄入的数量和质量与肌少症的发生和发展密切相关。增加老年人膳食中蛋白质的摄入量有利于促进肌肉蛋白质的合成，肌肉质量和肌肉力量的降低与膳食中较低的蛋白质摄入相关。一项随机开放性研究显示，膳食中蛋白质的摄入或通过蛋白质口服补充，使蛋白质摄入量在  $1.2\sim 1.5\mathrm{g / kg\cdot d}$  ，可改善四肢肌肉质量指数（appendicular muscle mass index，AMMI）[4]。因此，充足的蛋白质摄入是肌少症防治中的重要因素。

在临床中，老年人因消化功能、味觉的减退，加之1种或多种共病的存在，膳食中能量和蛋白质的摄入往往不足，故而口服营养补充剂是常用的补充能量和蛋白质的手段。近些年，多项研究评估了单独补充蛋白质、氨基酸或与其他营养素联合使用在肌少症防治中的作用。此外，补充蛋白质联合运动作为肌少症干

预手段的研究也有综述报道。多数随机对照研究中采用不同剂量的乳清蛋白、氨基酸或其代谢产物  $\beta$ - 羟基  $\beta$ - 甲基丁酸（HMB）]补充作为干预方法，研究中纳入的人群包括社区居住的健康老年人、肌肉力量减退的老年人、肌少症和衰弱的老年人。由于纳入人群地区、年龄、健康状态、干预方式或蛋白质补充剂量、干预周期的不同，结果显示出不一致性：大多数研究证实蛋白质/氨基酸的口服补充以及联合维生素D或抗阻运动可改善肌肉质量、肌肉力量或躯体功能（步速）。然而，也有研究未能观察到蛋白质/氨基酸的口服补充对肌肉质量或肌肉力量的有益作用。对于该问题的综述和荟萃分析显示：亮氨酸可增加老年人的肌肉蛋白合成；与单纯抗阻运动相比，蛋白质口服补充联合抗阻运动在预防老年人增龄相关的肌肉质量衰减和腿部力量降低方面可能具有更好的效果，尤其是在  $\mathrm{BMI} \geqslant 30.0$  的老年人中。此外，有研究显示，口服  $3 \mathrm{~g}$  HMB可产生等同于单独补充亮氨酸对肌肉蛋白合成的作用，同时可抑制腿部肌肉蛋白质的分解。然而，与其他氨基酸相比，服用HMB的益处尚存在争议，因为单独使用等量的小剂量亮氨酸（  $3 \mathrm{~g}$  ）几乎可最大限度地刺激肌肉蛋白的合成。蛋白质/必需氨基酸的合成代谢反应具有剂量依赖性和短暂性，目前普遍认为：通过摄入  $20 \sim 40 \mathrm{~g}$  的优质蛋白质如肉类、乳清蛋白或  $10 \sim 20 \mathrm{~g}$  的必需氨基酸可促成营养的最大合成代谢反应。当摄入的必需氨基酸超过用于肌肉蛋白合成底物的剂量时，循环中过量的氨基酸可直接被氧化和进入尿素合成途径，剩余的碳骨架可用于其他组织的糖异生和生酮。因此，尚需相关研究进一步确定干预剂量、周期。

# 2抗氧化剂和维生素D：对于血清25-羟维生素D水平不足的老年人，应补充维生素D制剂

近年来，关于抗氧化剂与肌少症防治的相关研究也在逐渐增加。其理论依据是骨骼肌中活性氧的过量产生会导致衰老过程中肌肉质量和肌肉功能的丧失，尤其是线粒体损伤刺激活性氧的堆积，这被认为可能是肌少症发展的关键。最近的荟萃分析显示：富含抗氧化剂的食物或补充剂可改善肌肉力量和功能，其中维生素E联合维生素D和蛋白质补充、镁、茶叶中的儿茶素，以及增加水果和蔬菜的摄入量是目前最优的选择。其中，25- 羟维生素D水平  $< 30 \mathrm{~nmol} / \mathrm{L}$  的群体补充维生素D对肌肉力量更为重要。此外，越来越多的证据表明，补充omega- 3多不饱和脂肪酸对患

有肌少症的老年人有益，可能会增加运动和（或）蛋白质补充的效果。然而，治疗和预防肌少症的确切剂量、频率和使用方法（单独或联合）仍需进一步探索。
