2000年美国精神病学会颁布的《精神疾病诊断与统计手册第四版修订版》（DSM- IV- TR）对谵

妄定义如下：谵妄是急性发作的意识混乱，伴注意力不集中、思维混乱、思维不连贯以及感知觉异常，之后该定义得到广泛的使用。2013年出版的美国《精神疾病诊断与统计手册第五版》（DSM- V）未对谵妄进行重新定义，直到2018年，世界卫生组织发布国际疾病分类第11版（ICD- 11）对谵妄进行了重新定义：谵妄是急性或亚急性起病的注意障碍（即指向、聚焦、维持和转移注意的能力减弱）和意识障碍（即对环境的定向力减弱），在1d内症状常出现波动，并伴其他认知障碍（如记忆、语言、视空间功能或感知觉障碍等），可影响睡眠觉醒周期，其病因常为非精神行为障碍类疾病、物质或某种药物中毒或戒断。相比较而言，ICD- 11关于谵妄的定义能够更全面地涵盖谵妄的病因、起病形式、临床表现，因此经专家研讨后，建议使用ICD- 11的谵妄定义。

# 一、谵妄的流行病学

由于评估方法、调查群体的不同，文献报道的谵妄发生率存在差异。国外文献报道，社区人群谵妄的患病率为  $1\% \sim 2\%$  ，综合医院  $17 \sim 95$  岁住院患者谵妄发生率为  $17.7\%$  ，卒中后谵妄发生率为  $32\% \sim 52\%$  ，由此可见综合医院住院患者谵妄的发生率显著高于社区群体[3,6- 7]。谵妄常见于老年群体，有报道急诊老年患者谵妄发生率为  $17.2\%$  [8]，住院老年患者谵妄发生率为  $10.0\% \sim 21.6\%$  [9- 10]，老年患者冠状动脉旁路手术后谵妄发生率达  $32\%$  [11]，髋关节术后谵妄发生率高达  $40.5\% \sim 55.9\%$  [12- 13]，而65岁以上ICU患者中，约  $70\%$  的患者出现谵妄[14]。

国内研究相对较少，不同疾病住院患者的谵妄发生率差异较大。有报道呼吸监护室患者谵妄患病率为  $8.93\%$  [15]；ICU患者谵妄的发生率为  $33.1\%$  [16]；老年患者髋部骨折术后谵妄发生率为  $30\%$  [17]；老年前列腺增生术后患者谵妄发生率为

$12.5\%$  [18]；老年胃肠道手术患者术后妄的发生率为  $25.4\%$  [19]；65岁及以上非心脏手术患者术后妄的发病率为  $11.1\%$  ，其中开颅手术术后妄发病率最高，达  $57.1\%$  [20]。

共识二：综合医院住院患者妄患病率高于社区人群，老年患者、重症患者、术后患者妄发生率较高。

# 二、妄的病因及危险因素

妄的发生常由多种因素引起，包括易患因素和触发因素[21]。

1. 易患因素：高龄、认知障碍、衰弱、药物/酒精依赖、听力或视力障碍、罹患多种躯体疾病等是常见的易患因素[22]。其中认知障碍的影响最明显，认知障碍程度越重，发生妄的风险越高。

2. 触发因素：妄的触发因素包括脑部疾病、其他系统性疾病、环境因素及药物因素。（1）脑部疾病：包括脑外伤、脑卒中、硬膜下血肿、脑炎、癫痫等。（2）其他系统性疾病：包括呼吸系统疾病（低氧或二氧化碳增高）、营养及代谢疾病（贫血、叶酸、维生素B1、B12缺乏、低血糖、脱水、电解质紊乱、酸中毒）、心血管疾病（低血压、心肌梗死）、感染（泌尿系、肺部、关节、瓣膜等部位感染）、便秘或泌尿系统疾患和操作（尿潴留、导尿等）、外伤（如骨折）、手术和麻醉、中毒或戒断（酒精、毒品）、疼痛等。（3）环境因素：噪音、活动受限、居住环境改变、情感打击等。（4）药物因素：以下药物会增加妄的发生风险：阿片类药物、苯二氮草类药物、非苯二氮草类安眠药物、抗组胺药、二氢吡啶类药物、  $\mathrm{H}_{2}$  受体拮抗剂、部分抗精神病药物、三环类抗抑郁药、抗帕金森病药物等[23-24]。

共识三：妄是多种因素导致的神经精神综合征，高龄、认知障碍、衰弱、视听障碍是妄常见的易患因素，而脑部疾病、其他系统性疾病、环境因素以及药物因素均可诱发妄的发生；故对妄的患者，需积极查找并处理触发因素。

# 三、病理生理机制

妄的病理生理机制复杂且尚未充分了解。因为妄的病因及危险因素众多，临床表现多样，故难以用单一的病理生理机制来解释妄的发生和发展过程，推测可能多种病理生理机制与妄的发生有关，如神经炎症机制、神经老化、氧化应激、神经递质的失衡、神经内分泌紊乱、褪黑素调节障碍等，不同机制互相补充、部分互有重叠，最终产生神经递质调节障碍和神经网络连接障碍，导致系统

整合衰竭，从而出现妄症状。其中神经递质失衡机制（如乙酰胆碱、褪黑素缺乏；多巴胺、去甲肾上腺素、谷氨酸增多；  $\gamma -$  氨基丁酸、天冬氨酸、五羟色胺的增多或减少)学说在妄的发病机制中具有重要作用，是妄药物治疗的理论依据[25]。

共识四：妄的病理生理机制复杂，可能多种神经递质失衡参与其中，是妄药物治疗的理论依据。

# 四、妄的临床表现及分型

妄的基本特征是急性或亚急性注意力和意识状态的障碍，可以表现为与基础水平明显不同的意识内容损害（注意力下降、定向力差）和意识水平损害（觉醒程度损害），也可有语言障碍、感知及行为障碍，多呈现波动性病程，一般持续数小时至数天[26]。

妄主要有以下核心表现：（1)注意力障碍：表现为定向、聚焦、持续和变更注意力的能力下降，患者的注意力分散，容易被无关刺激干扰，不能根据询问内容恰当回答或转换话题；（2）意识内容障碍：表现为对环境的定向力减弱，有时对自身状态（如姓名、年龄、职业等）的定向力减弱，同时伴有觉醒程度下降、淡漠、嗜睡等意识活动降低的表现，或警醒、易激惹、烦躁、有攻击性和拒绝配合诊疗活动等意识状态过度增强的表现；（3）此障碍在很短的时间内发展，通常为数小时至数天，倾向于在1d内波动，在傍晚和夜晚时加重；（4）可伴发认知功能障碍，主要包括感知觉障碍（如错觉或幻觉）、记忆和学习障碍、抽象思维及理解能力障碍、执行功能障碍（即确定目标、制定和修正计划、实施计划、进行有目的活动的能力）、语言障碍；（5）生物节律、情绪调节障碍，其特征是睡眠觉醒周期紊乱、睡眠倒错、恐惧、易怒、易激惹、焦虑不安[26]。

目前妄可分5个临床亚型[27]：（1）活动亢进型：患者表现高度警觉、烦躁不安、易激惹、可有幻觉和妄想、有攻击性精神行为异常，是最容易被发现的一种类型；（2）活动抑制型：表现为睡眠增多，表情淡漠、语速及动作缓慢，因症状不易被察觉，常漏诊；（3）混合型妄：表现为上述两种妄类型交替出现，反复波动；（4）亚综合征型（subsyndromal type）：表现为部分妄症状，只符合部分妄诊断标准，常被忽视；（5）迁延型或持续型妄：相对较少，多见于既往存在认知功能障碍的患者，或妄继发于颅内新发病变者。

共识五：妄为急性或亚急性起病的伴有波动

性的注意力和意识内容障碍，往往伴有其他认知障碍和生物节律及情绪障碍；妄可分为5个临床亚型，其中活动亢进型最易被发现，混合型妄相对较易发现，活动抑制型及亚综合征型常被忽视，迁延型相对较少见。

# 五、辅助检查

大部分妄经详细的病史询问及床旁认知检查可初步明确诊断。辅助检查主要用于提高识别率、寻找触发因素。

（一）妄是一个临床综合征，筛查量表的合理使用可以提高识别率

1. 意识模糊评估量表（CAM）：CAM是目前使用最广泛、最有效的妄筛查工具[28-29]，改良版根据《美国精神障碍诊断与统计手册第3版》（DSM-Ⅲ）妄的诊断标准于1990年编制，适用于非精神心理专业的医生护士筛查妄。CAM针对妄的4个特征分别对应4个问题条目：（1)急性发病和精神状态的波动性变化；（2)注意力集中困难；（3)思维混乱；（4)意识状态的改变；诊断必须满足(1)和(2)这2条，且至少满足(3)或（4)其中的1条。与老年医学专家、精神科专家、神经心理专家基于DSM-Ⅲ、DSM-IV及ICD-10做出的妄诊断相对比，CAM具有较高的敏感性（  $94\% \sim$ $100\%$  )和特异性  $(90\% \sim 95\%)[30]$  。为保证结果的准确性，检查者需经培训后才可以使用该量表。在CAM量表的基础上，还衍生出重症监护室意识模糊评估量表(CAM-ICU)[31]，适合患者无法言语配合时使用；中文版  $3\mathrm{min}$  省妄诊断量表(3D-CAM)应用于老年患者，敏感度为  $94.73\%$  、特异度为  $97.92\%$  可用于评估内外科老年妄患者[32]。对于深度镇静或昏迷患者不能进行妄评估。

2.4A测试（4“A"sTest，4AT）：4AT对警觉性、定向力、注意力、急性改变和波动性病程等4个方面进行评估。评分  $\geq 4$  分为可能妄合并/不合并认知损害，  $1\sim 3$  分为可能认知功能损害，0分为无妄或无严重认知功能损害（确保第4项问题完成的前提下）[33]；其敏感性为  $90\%$  、特异性为 $84\%$  。4AT目前有12种语言版本（含简体中文版本），经多中心验证，简便易行，不需要特殊培训即可进行。

3. 省妄评估量表（DRS）：临床应用也较广泛，最早于1988年发表[34]后为增加其可重复性，1998年进行修订，适用于经过精神科培训的临床医师，共16个条目，2009年妄评估量表-修订版（DRS

R- 98)发表中文版，并进行信效度检验，划界分15.5分，灵敏度为  $89.3\%$  、特异度为  $96.8\% [35]$  0

除以上量表之外，尚有其他筛查工具，但应用相对较少，未进行大样本信效度评价，如简短意识模糊评价量表（bCAM）[36]、记忆妄评估量表(MDAS)[37]等。

妄具有波动性的特点，需进行动态的评估，推荐对可疑妄患者或妄患者每日进行感知觉评价以全面评估病情[22]。衣摸床、虚空理线等异常手部动作是妄特异性的临床表现，在临床工作中需加以关注[38]

# （二）触发因素的筛查

妄是多种原因导致的神经精神综合征，对妄患者需行一系列辅助检查以明确触发因素，有助于及早开展相关治疗、改善预后。以下检查是妄患者需要常规进行的：（1)血常规、肝肾功能、血糖、血电解质、动脉血气分析、体液平衡、C- 反应蛋白；(2)尿常规，血、尿及体液培养；（3)胸片、心电图等；（4)根据病史及体质检查线索提示的特异性检查以明确病因或触发因素[21]。

共识六：推荐CAM、4AT、DRS用于妄评估；对未经精神科培训的医护人员，推荐采用4AT量表进行评估，对经过相关培训的人员推荐使用CAM、CAM- ICU和DRS- R- 98；对妄患者，需进行相应辅助检查，以明确病因或触发因素。

醇可显著降低老年妄患者DRS评分，改善临床整体印象评分[44]；但另一项RCT研究结果显示，个体化治疗妄诱因及支持治疗可改善姑息治疗伴妄患者的妄评估量表评分，缩短妄持续时间，疗效优于氟哌啶醇[45]。对6项RCT研究进行Meta分析，结果显示，氟哌啶醇未能降低ICU妄患者的短期死亡率、ICU住院天数及妄持续时间[46]，提示氟哌啶醇可能减轻妄症状，但对长期预后无明显改善。2项安慰剂对照RCT研究提示，痤疮可缩短妄持续时间、改善妄的非认知症状，但受限于研究样本量偏小、偏倚较大，不足以据此进行高级别的治疗推荐[47- 48]。2项以氟哌啶醇为对照药物的RCT研究提示，痤疮平控制妄症状疗效与氟哌啶醇相当[49- 50]。1项安慰剂对照RCT研究提示臭氮平可改善妄症状[44]。3项小样本RCT研究提示臭氮平和利培酮可改善妄症状，疗效与氟哌啶醇相当[50- 52]。1项RCT研究提示，利培酮未能改善姑息治疗患者的妄症状评分[53]。妄药物治疗效果欠佳，且治疗相关不良反应的风险增加，故妄治疗以触发因素治疗及非药物治疗为主。若出现以下情况，考虑使用抗精神病药物，其治疗目的为镇静、控制兴奋躁动和精神病性症状：（1)妄伴行为及情感障碍如兴奋、激越、行为紊乱、错觉、幻觉和妄想等导致患者极度痛苦；（2)危及患者或他人安全；（3)干扰基本的检查及治疗；（4)非药物治疗无效时。治疗药物推荐氟哌啶醇、痤疮平、臭氮平及利培酮，以上药物宜自小剂量开始，根据妄改善情况及不良反应逐渐增加剂量；一般治疗  $1\sim 2$  周，妄消失2d后可逐渐停药。用药期间需监测锥体外系不良反应、心电图QT间期及意识水平的改变，治疗后若妄症状仍不改善，建议重新评估妄的诱因并予以治疗，或随访判断是否存在痴呆。

共识八：妄治疗以触发因素治疗为主，对症治疗首选非药物治疗。大部分妄症状尤其是活动抑制型妄的症状可以通过非药物治疗得到改善，不推荐对妄患者常规使用抗精神病药物。对妄伴行为及情感障碍导致患者极度痛苦、危及患者或他人安全、干扰基本的检查及治疗，且非药物治疗无效时，可使用抗精神病药物进行治疗。治疗药物推荐氟哌啶醇、喹硫平、奥氮平及利培酮。以上药物宜自小剂量开始，根据妄改善情况及不良反应逐渐增加剂量，一般治疗  $1\sim 2$  周，妄消失2d后可逐渐停药。

# 八、预防

预防是妄干预的核心问题，在高危群体中尤其重要。妄的预防可以分为非药物及药物预防两个方面。

1. 谷高危个体的识别：2019年英国国家临床医学研究所（NICE）建议，对新入院患者需进行妄风险评估，以下危险因素中有任何一个存在，即为妄高危个体：（1)65岁以上；（2)轻度认知功能障碍或痴呆，若认知障碍不肯定，需用标准化的评估手段进行认知功能评价；（3）新发髋部骨折；（4）重症疾病。

2. 谷妄的非药物预防：很多躯体疾病均可导致妄，故需对不同危险因素进行多重干预，包括制定睡眠计划、处理容量不足、改善视听觉、尽早康复锻炼、训练定位能力、减停不必要的药物、评估并

处理疼痛等。1项系统性回顾研究提示，无论是内科住院患者还是外科住院患者，与常规护理比较，对妄的危险因素进行多重干预可显著降低妄的发生率[54] [相对危险度  $(RR) = 0.69,95\% CI$ $0.59\sim 0.81]$  ；另1项系统性回顾研究结果也显示，围术期多重干预可降低择期非心脏手术术后妄的发生率  $(RR = 0.44,95\% CI:0.25\sim 0.78)$  。

3. 谷妄的药物预防：（1)抗精神病药物预防性使用不能缩短妄的病程、严重程度、住院时间及ICU住院时间，也不能降低妄的发生率[54,56]  $(RR = 0.73,95\% CI:0.33\sim 1.59)$  。（2)胆碱酯酶抑制剂未能降低妄的发生率[54]  $(RR = 0.68$ $95\% CI:0.17\sim 2.62)$  。（3)系统性回顾未发现褪黑素或褪黑素受体激动剂雷美替胺可降低妄的发生率[54]  $(RR = 0.41,95\% CI:0.09\sim 1.89)$  。（4)一项RCT研究发现食欲素受体拮抗剂苏沃雷生可降低ICU或急症病房患者妄的发生率[57]  $0\%$  比  $17\% ,P = 0.025)$  ；另1项RCT研究也发现苏沃雷生有降低ICU患者妄发生率的倾向，但差异无统计学意义  $(14.7\%$  比  $33.3\% ,P = 0.069)$  但该研究发现苏沃雷生可降低亚临床妄的发生率  $(17.6\%$  比  $47.2\% ,P = 0.011)$  ，推迟妄出现[58]；2项研究结果不一致，其对妄预防的疗效和安全性尚需开展高质量RCT研究。（5）脑电双频谱指数指导麻醉可降低术后妄的发生率[54]  $(RR = 0.71,95\% CI:0.60\sim 0.85)$  。（6)一项系统

性回顾研究提示，右旋美托咪啶可降低谵妄的发生率  $(RR = 0.57,95\% CI:0.34\sim 0.87)$ ，可用于全身麻醉或机械通气时的镇静治疗[54]。

共识九：推荐对新入院患者进行谵妄的风险评估；推荐非药物预防策略，不推荐抗精神病药物及胆碱酯酶抑制剂预防性使用；对需要全身麻醉的患者推荐脑电双频指数监测麻醉，控制麻醉深度；对全身麻醉或机械通气并需要镇静的患者，推荐右旋美托咪啶镇静治疗。
