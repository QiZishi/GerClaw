# 社区晚期肿瘤患者便秘居家管理中国专家共识（2025版）

便秘是晚期肿瘤患者的第三大常见症状，仅次于疼痛和厌食，严重影响患者的生活质量，在晚期肿瘤患者中，便秘的患病率达  $60\%$  ，其中服用阿片类药物的晚期肿瘤患者的便秘患病率可能高达  $87\%$  [1- 2]。而社区晚期肿瘤居家患者因液体和营养摄入不足、活动受限、疼痛、药物不良反应及多病共存等因素，便秘发生风险更高[3- 4]。为充分利用社区可及资源，提升社区晚期肿瘤居家患者便秘管理效率与生活质量，中国康复医学会社区康复工作委员会组织肿瘤学、姑息医学、护理学以及社区康复等领域专家，基于循证证据和国际临床实践指南，聚焦社区晚期肿瘤居家患者群体，整合“社区- 家庭”协作管理模式，构建科学、实用的便秘管理体系，从而全面优化便秘管理流程。本共识面向社区医护人员及社区晚期肿瘤居家患者群体，强调以患者主观感受为核心，兼顾非药物与药物干预，并提供远程协作支持与紧急转诊，以期为社区医护人员及家庭照护者提供可操作性强的参考框架，制订适用于社区及家庭的社区晚期肿瘤居家患者便秘管理方案。

# 1 共识制订方法

本共识由中国康复医学会社区康复工作委员会联合复旦大学附属华山医院康复医学科于2025年1月发起，并于2025年3月正式启动编写。经多轮讨论，编写小组明确本共识聚焦便秘居家管理风险评估、社区

居家干预方案、“社区- 家庭”协作管理等核心内容。

系统检索PubMed、Web of Science、Cochrane Library、中国期刊全文数据库、万方学术期刊全文数据库、中国生物医学文献数据库和维普中文科技期刊数据库以及美国国家综合癌症网络、欧洲肿瘤内科学会、世界卫生组织等专业网站。采用主题词和自由词相结合的检索策略，明确相应中文检索词及英文检索词进行系统检索，文献纳入类型包括指南、专家共识、随机对照试验、系统评价、观察性研究等，并进一步对纳入的文献进行质量评价。

依据循证证据与质量评价结果，由编写小组结合我国临床实际情况与康复资源，拟定共识初稿，对尚缺乏高质量证据的问题，组织召开多轮专家论证会形成专家意见，编写小组在充分吸纳专家指导建议并反复修订后形成本共识。

# 2 便秘居家管理风险评估

欧洲肿瘤内科学会临床实践指南建议对所有晚期肿瘤患者定期进行便秘评估，以便尽早识别并进行干预[1]。一旦发生便秘，应评估发生原因和严重程度，积极治疗粪便嵌塞、肠梗阻等[3]。

# 2.1 评估方法

2.1.1 罗马IV标准的居家应用 罗马IV标准旨在筛查功能性胃肠病，可作为临床试验的病例纳入标准[5]

但尚未在晚期肿瘤患者中得到验证[6]。根据罗马IV标准，阿片相关便秘定义为“开始使用或增加使用阿片类药物后新发或加重的便秘”，需满足以下至少2项[7]： $①\geq 25\%$  的时间排便费力；  $(2)$  干硬粪便（Bristol粪便性状量表分型为  $1\sim 2$  型)；  $(3)$  有排便不尽感；  $(\widehat{\Delta})$  需要手法辅助排便。

国际癌症支持治疗协会指南建议采用多维度评估框架，涵盖病因筛查（如药物、脱水、活动受限）粪便性状（Bristol粪便性状量表分型为  $1\sim 2$  型）及排便频率（  $\leq 3$  次/周）等核心指标[8]。社区医护人员可结合罗马IV标准与Bristol粪便性状量表，快速评估并识别具有便秘高风险的社区晚期肿瘤居家患者。

2.1.2 肠功能指数评估 肠功能指数是由排便容易程度、是否有排便不尽感和便秘主观判断3项内容组成，可用于评估社区晚期肿瘤居家患者的阿片相关便秘[9- 10]。

2.1.3 患者主观评估 已有研究证实，患者主诉症状（如排便困难、需依赖泻药）与客观指标（如排便频率、末次排便间隔时间）显著相关，且均可有效提示便秘存在，其中因主观评估可直接影响患者生活质量，其临床价值更优于客观指标[11]。因此，在社区晚期肿瘤居家患者有独立交流能力时，需要更注重患者的主观感受。肿瘤护理论坛相关指南强调患者主诉在便秘居家管理风险评估中具有重要作用，推荐使用排便日志量化记录社区晚期肿瘤居家患者的每日排便次数、费力程度（采用视觉模拟评分法进行评估）；粪便硬度（采用Bristol粪便性状量表进行评估）及是否需要手法辅助；腹胀、腹痛等伴随症状的频率与强度等[12]。

# 2.2 社区医护人员入户评估

2.2.1 病史采集 完整的病史有助于确定便秘原因，规范的问诊流程应涵盖以下核心要素，主要包括饮食习惯、排便频率、药物使用情况、体力活动水平、是否存在相关胃肠道疾病、多病共存情况以及环境因素等[1]。

2.2.2 腹部体格检查 社区医护人员应对患者腹部进行听诊、触诊，重点排查粪便嵌塞、肠梗阻等[13]。腹部体格检查有助于区分阿片相关便秘与其他器质性便秘（如肿瘤压迫、代谢紊乱等导致的便秘）[14]。

2.2.3 紧急转诊 欧洲肿瘤内科学会指南建议，当患者出现以下情况时需要立即转诊：  $①$  持续性腹痛伴呕吐（疑似肠梗阻）；  $(2)$  便中带血伴发热（需要排除肠道穿孔或感染）；  $(3)$  突发腹胀加重伴肠鸣音消失（警惕肠麻痹或机械性梗阻）[1]。

# 3 社区居家干预方案

# 3.1 预防性干预措施

3.1.1 预防性给药 基于阿片类药物对肠道运动的

抑制效应（减少肠道动力，增加括约肌张力），需要在用药早期启动泻药干预来预防便秘发生[15]。所有接受阿片类药物治疗的社区晚期肿瘤居家患者，应在用药初期即预防性使用渗透性泻药（如聚乙二醇）或刺激性泻药（如比沙可啶）[7,16]。欧洲肿瘤内科学会指南强调，阿片相关便秘的预防应优先于治疗，渗透性泻药因其安全性高、长期使用不损伤肠神经，可作为一线选择[1,15]。

3.1.2 健康教育 社区医护人员需要强调晚期肿瘤患者便秘居家管理的重要性，对患者及家属进行健康教育，使其提前了解便秘风险[15]。

# 3.2 非药物干预的居家管理

3.2.1 腹部按摩 腹部按摩是治疗便秘的有效方法，欧洲肿瘤内科学会指南指出，腹部按摩可减轻胃肠道症状，提高肠道效率[1]。Yildirim等[17]的研究显示，规律腹部按摩可使排便频率增加  $13\%$  ，并可改善粪便硬度。腹部按摩时，患者取仰卧位，膝盖微曲，家属手掌涂上温橄榄油，以脐为中心顺时针绕圈，注意压力适中，  $15\mathrm{min}/$  次，2次/d，1次在早餐前  $30\mathrm{min}$  ，1次在晚餐前  $30\mathrm{min}$  ，每次按摩后热敷 $10\mathrm{min}^{[17 - 18]}$  。需要注意的是，腹部按摩需要在患者的耐受范围内进行，并且确保患者能够配合[17]。对于有腹部或盆腔肿块的患者，腹部按摩可能会导致肠梗阻或其他并发症，因此在进行腹部按摩之前还需要对患者进行全面评估，以排除其他可能的风险因素[1]。

3.2.2 饮食调整 美国国家综合癌症网络临床实践指南指出，便秘患者采用高纤维饮食联合充足水分摄入可软化粪便[3]。世界胃肠病学组织全球指南推荐便秘患者进行饮食调整，具体方案如下：  $①$  规律进食早餐；  $(2)$  进食富含膳食纤维的食物；  $(3)$  选择富含果胶的水果，如苹果、草莓、荔枝或梨，通过增加粪便体积来通便；  $(\widehat{\Delta})$  摄入含有益生菌的发酵奶制品； $(5)$  摄入2L饮用水，减少摄入含咖啡因、酒精和高糖的饮料；  $(\widehat{G})$  减少摄入动物脂肪含量高的油炸食品； $(\Xi)$  减少摄入精制糖；  $(\widehat{G})$  调整饮食时间和饮食习惯[19]。3.2.3 卧床活动方案 除被动腹部按摩外，有研究显示，卧床患者每日规律进行踝泵运动（10次/h）联合侧身翻身（1次/2h）可有效减少肠道淤滞，改善排便频率[13,17]。此外，排便私密性、排便姿势以及最大限度发挥自然的晨起直肠排便反射等均可促进自然排便[13]。

# 3.3 中医适宜技术的社区应用

已有研究显示，针对神阙穴和足三里穴进行大黄、木香等中药贴敷，  $6\mathrm{h}/$  次，1次/d，连续7d，可显著改善中晚期肿瘤患者的便秘症状，且安全性及患者的依从性均较好[20]。荣丽媛等[21]的研究表明，联合应用多种中医适宜技术可改善阿片相关便秘，有效缓解患者的不适症状，提高患者的生活质

量。社区医护人员可积极引入相关中医适宜技术，针对社区晚期肿瘤居家患者进行个性化指导。

# 3.4 社区阶梯式药物管理

社区阶梯式药物管理主要包括渗透性泻药优先使用原则和阿片相关便秘的特异性管理，在社区晚期肿瘤居家患者的便秘管理中均起着重要作用。

3.4.1 渗透性泻药优先使用原则 深圳肿瘤内科学会指南指出，聚乙二醇可作为一线渗透性泻药，安全性优于刺激性泻药，长期使用不损伤肠神经，乳果糖 $(10\sim 30\mathrm{mL / d})$  可作为替代选择，但需注意腹胀风险[1]。3.4.2 阿片相关便秘的特异性管理 对于传统泻药治疗无效的难治性阿片相关便秘，推荐甲基纳曲酮或纳洛酮[1]。已有研究显示，皮下注射甲基纳曲酮 $(0.15\sim 0.30\mathrm{mg / kg}$  或固定剂量  $8\sim 12\mathrm{mg}$  可在  $24\mathrm{h}$  内快速缓解便秘，连续使用2周效果显著且安全性较好[2]。但皮下注射甲基纳曲酮需要社区医护人员操作，基于居家环境条件受限，优先推荐患者使用甲基纳曲酮口服制剂或羟考酮/纳洛酮复方制剂[1]。相关研究显示，羟考酮/纳洛酮复方制剂可显著降低肠道功能指数[7]。对长期使用阿片类药物的社区晚期肿瘤居家患者，初始治疗时建议联合使用聚乙二醇与刺激性泻药（如比沙可啶），但后者需限制使用频率（ $\leq 3$  次/周），以避免肠道依赖[23]。

# 4 “社区-家庭”协作管理

“社区- 家庭”协作管理可有效提升便秘管理效率，已有研究证实，可通过制订延续性护理方案，进行定期随访及健康教育，显著改善患者的肠功能指数[24]。应明确社区医护人员以及家属的角色分工，如由社区医护人员优先推荐安全性高的渗透性泻药，明确警示刺激性泻药的使用频率，提供腹部按摩指导，为家属制作可操作的腹部按摩步骤图解，设计家庭版排便日志，并进一步告知家属紧急转诊指征；社区护理人员还应定期入户评估，进行腹部触诊，指导家属协助观察患者腹部体征、进行腹部按摩操作、学习记录排便日志，此外，社区医护人员还应建立便秘管理常备药物清单，涵盖聚乙二醇、乳果糖等一线药物。配备简易灌肠工具包以供紧急使用[1,3,24]，并进一步构建远程协作机制，实现临床医师远程指导、社区医护人员入户评估及家属执行的协同管理[24]。

# 5 小结

基于循证证据和国际临床实践指南，本共识系统构建了社区晚期肿瘤居家患者便秘管理标准，通过整合便秘居家管理风险评估方案以及社区居家干预方案，实现了对患者便秘风险的动态识别与干预。本共识强调以患者的主观感受为核心评估依据，兼顾非药物与药物干预措施，推荐预防性使用渗透性泻药或刺

激性泻药来有效降低阿片相关便秘发生率，采用中医适宜技术和腹部按摩等非药物干预手段来明显改善患者的症状，提高患者的生活质量，此外，“社区- 家庭”协作管理模式还可进一步提高管理效率。本共识为社区医护人员及家属提供了科学且可行的操作框架，尤其适用于资源有限情况下的居家管理，对优化社区晚期肿瘤患者的生存质量具有重要临床意义。未来还需进一步开展更多本土化研究，以验证社区居家干预方案的有效性，并探索智能化远程监测技术的综合应用潜力。本共识基于当前最佳循证证据制订，随着学科发展，编写小组将通过定期评估对本共识进行动态更新，保持共识时效性。

中国康复医学会社区康复工作委员会

尿失禁是指无法自主控制排尿，导致尿液不自主流出[1]。据一项全国范围内的大型调查显示，2021年中国女性尿失禁的加权患病率为  $16.0\%$  ，其中压力性尿失禁是最主要的亚型，占  $7.0\%$  ，混合性尿失禁和急迫性尿失禁分别占  $6.5\%$  和  $1.9\%$  [2]。已有研究表明，我国北方农村地区65岁及以上老年人群的尿失禁患病率达  $9.3\%$  [3]。老年尿失禁常与活动能力下降、认知功能障碍以及其他合并症同时出现，需要采取特定干预措施[4]。为进一步优化老年尿失禁患者的全周期康复管理路径，充分发挥基层医疗卫生机构在老年尿失禁筛查、评估及康复治疗等环节中的关

键作用，强化社区医师对老年尿失禁评估、老年尿失禁居家康复管理的认知深度与重视程度，全方位改善社区老年尿失禁患者的生活质量，中国康复医学会社区康复工作委员会组织编写小组，结合我国临床实际情况与康复资源，依据循证证据制订了本共识[5]。
