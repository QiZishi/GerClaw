中国冠心病患病率和死亡率仍处于上升阶段[1]。冠心病可被归类为急性冠状动脉综合征(ACS)或慢性冠状动脉综合征(CCS)[2]。CCS是老年冠心病患者的主要表现类型，患病人数多、病程长、预后差，导致生活质量下降，疾病负担与经济负担沉重；尤其是高危CCS患者，病死率和总死亡率高。尽管目前标准的预防和治疗策略，使患者生存期得以延长，但残留心血管风险仍然较高[3]。抗栓治疗是冠心病综合管理的重要策略之一，对于老年CCS高危患者来说，更安全、更适合的抗栓策略尤为重要[2-3]。目前针对老年CCS高危人群的指南或共识较少，为更好地规范老年CCS高危患者的抗栓治疗，中华医学会老年医学分会组织国内专家，依据已发表的临床研究证据，结合国内外相关指南、共识及专家建议，特制订本共识，供临床医生参考。

# 一、老年CCS高危患者定义

# （一）老年CCS患者定义

老年CCS患者是指65岁及以上除ACS以外的冠状动脉疾病状态[3]，包括以下六种最常见的临床情况[2]：（1)冠状动脉疾病伴有稳定的心绞痛症状和(或)呼吸困难症状；(2)新发心衰或左心室功能障碍，且可疑为冠状动脉疾病；(3)ACS或冠状动脉血运重建后1年内，无症状或症状稳定；(4)初次诊断或血运重建后超过1年，有或无症状；(5)可疑因血管痉挛或微血管病变而有心绞痛症状；(6)体检或筛查发现的无症状患者。老年CCS高危患者包括老年CCS缺血高危及老年CCS出血高危的临床情况。

# （二）老年CCS缺血高危患者定义

CCS高缺血风险定义为心血管死亡率  $>3\%$  年，低缺血风险定义为心血管死亡率  $< 1\%$  年[2]。各种临床合并症、复杂的冠脉解剖因素、介入器械相关因素均能增加缺血相关的心血管不良事件风险。老年CCS缺血高危患者包括冠状动脉多支病变、多血管床病变（缺血病变同时涉及冠状动脉、脑动脉、外周动脉）、多重高危因素（如糖尿病、高血压、血脂异常、肾功能不全等)及高龄  $(\geq 80$  岁）患者[2-4]。老年CCS高缺血风险特征见表1。

# （三）老年CCS出血高危患者定义

高出血风险与远期死亡率密切相关。CCS 高出血风险定义为 1 年出血学术研究联盟 (BARC)  $3 / 5$  型出血风险  $\geqslant 4\%$  或 1 年颅内出血风险  $\geqslant 1\%$ [5]。各种导致出血倾向的重要脏器基础疾病、合并用药及手术相关因素均可导致临床出血事件（表 2）。老年 CCS 高出血风险评估方法详见第五部分。

# 二、老年CCS缺血高危患者抗栓治疗

# （一）老年CCS缺血高危患者抗栓治疗原则

抗栓治疗可以使冠心病患者明显获益，对于老年CCS高危患者尤其如此[2-3]。缺血和出血风险评估是抗栓治疗的关键环节。

CCS患者抗栓治疗原则：（1)CCS患者在启动抗栓治疗前应充分评估缺血和出血风险以决定治疗策略[2,10]。（2)缺血高危且非出血高危患者推荐采用双联抗血小板治疗(DAPT)或双通道抗栓治疗(DPI)。（3)非缺血高危和(或)出血高危患者需避免或减少强化抗栓治疗(图1，表  $3)^{[2,10]}$  。（4）个体的遗传性状影响某些药物的反应性，CYP2C9和VKORC1基因多态性与华法林出血不良反应相关，CYP2C19基因多态性与氯吡格雷反应相关，有

选择地进行基因多态性监测，有助于精准评估个体情况，合理选择药物，但不推荐常规应用[4]。

# （二）老年CCS合并冠脉多支病变患者的抗栓治疗方案

1. 二级预防长期抗栓治疗：阿司匹林作为单联抗血小板治疗(SAPT)在减少CCS患者主要不良心血管事件(MACE)方面有明显的获益[11]。CAPRIE研究显示氯吡格雷单药与阿司匹林相比心血管二级预防轻度获益，主要由外周动脉疾病(PAD)患者亚组获益驱动，而安全性相似[12]。最新HOST-EXAM研究显示，对于行药物洗脱支架(DES)置入术后DAPT治疗  $6\sim 18$  个月无事件的稳

对于高缺血风险CCS患者，在当前标准二级预防下残余心血管风险仍较高，每年心血管死亡率仍可能高达  $3\%^{[2]}$  。越来越多的研究显示，对于老年高缺血风险CCS患者，长期阿司匹林和  $\mathrm{P2Y}_{12}$  受体拮抗剂的DAPT与阿司匹林单药相比有明显获益（表4)[14-16]。阿司匹林与利伐沙班  $2.5\mathrm{mg}$ ，2次/d组成的DPI用于CCS患者显著降低MACE，且具有明显的临床净获益[17-18]。

推荐1：老年CCS合并冠脉多支病变患者属于缺血高危人群，如无出血高风险因素，推荐在阿司匹林  $75\sim 100~\mathrm{mg / d}$  基础上加用第2种抗栓药物（  $\mathbf{P}2\mathbf{Y}_{12}$  受体拮抗剂或利伐沙班  $2.5\mathrm{mg},2$  次/d）；如存在出血高风险因素，推荐单一抗血小板治疗。

2.择期PCI围术期及二级预防抗栓治疗：在选择侵入性治疗策略时，推荐尽可能使用桡动脉入路以减少入路并发症[19-20]。与裸金属支架(BMS)相比，对老年患者使用药物洗脱支架(DES)与缩短DAPT疗程相结合，具有显著的安全性和疗效获益[19-20]

推荐2：针对目前药物球囊的使用越来越广泛，推荐在血管直径  $< 3.0\mathrm{mm}$  或者支架内血栓的

患者考虑使用药物球囊，在使用药物球囊的患者可考虑将DAPT时间减少至4周，但需持续监测及评估出血与血栓风险，随时调整DAPT的治疗时间。

推荐3：对BMS置入术后患者，推荐DAPT至少4周。

推荐4：对于DES置入术后患者，推荐DAPT治疗6个月，高出血风险患者可考虑缩短DAPT疗程至  $< 6$  个月，缺血高危、出血低危患者DAPT可维持6个月以上。

推荐5：对于药物球囊治疗患者，推荐DAPT至少4周。

3. 择期冠状动脉旁路移植术(CABG)围术期及二级预防抗栓治疗：接受择期CABG的CCS患者，通常应继续服用阿司匹林，并且其他抗血栓药物应根据其作用持续时间和适应证间隔停药（术前普拉格雷停药  $\geq 7$  d；氯吡格雷  $\geq 5$  d；替格瑞洛  $\geq 3$  d）；CABG手术后重新使用阿司匹林可改善移植物通畅性。小样本随机对照试验显示，对于非体外循环CABG患者，DAPT与阿司匹林单药相比，移植血管通畅率稍占优，但减少死亡率方面并不占优。COMPASS研究中，对1448例CABG后  $4\sim$  14d的患者进行亚组分析，提示DPI降低MACE获益与总体人群一致，该结果有待针对CABG人群做进一步探索与验证。[2,21-25]

推荐6：CCS患者CABG术后  $6\sim 24\mathrm{h}$  给予抗血小板药物治疗。

# （三）老年CCS合并多血管床病变

多血管床病变是指冠脉病变基础上合并至少有主动脉、脑动脉、胃肠道、下肢、上肢和肾脏血管床之一的病变。累及2个及以上血管床病变属于血栓栓塞高风险患者，宜进行强化抗栓治疗。现有研究显示，对于多血管床病变患者，DAPT或DPI可降低MACE，推荐在阿司匹林基础上联合  $\mathrm{P2Y}_{12}$  受体拮抗剂或利伐沙班  $2.5\mathrm{mg},2$  次/d用于合并多血管床病变且低出血风险的老年CCS患者的二级预防[16-17,28]。

（四）老年CCS合并其他高危因素

老年CCS合并其他高危因素，如糖尿病、高血压、慢性肾脏病、心衰及高龄时，应根据缺血及出血风险选择抗栓治疗方案。

三、老年CCS合并房颤患者的抗栓治疗

（一）老年CCS合并房颤患者的抗栓治疗原则

CCS患者需要进行抗血小板治疗以减少心肌缺血事件，而血栓栓塞高风险的房颤患者则需口服抗凝药物(OAC)治疗以减少卒中等血栓栓塞事件。CCS与房颤合并存在时，联合应用抗血小板与抗凝治疗可有效减少缺血及血栓栓塞事件，但增加出血风险[6]。对于老年CCS合并房颤患者，根据缺血/血栓栓塞和出血风险评估结果考虑选择OAC单药或OAC+SAPT治疗。高龄患者常合并肝肾功能不全及同时使用多种药物，可能增加药物间相互作用及不良反应风险，建议对高龄患者加强综合管理，并适当调整OAC剂量[6]。

# （二）抗栓治疗前评估：

1. 血栓栓塞风险评估：非瓣膜性房颤(NVAF)患者推荐采用  $\mathrm{CHA}_2\mathrm{DS}_2$  -VASc评分进行血栓栓塞风险评估[6,41]：(1)  $\mathrm{CHA}_2\mathrm{DS}_2$  VASc评分  $\geq 2$  分（男）/  $\geq 3$  分（女）的患者应进行长期抗凝治疗；(2)  $\mathrm{CHA}_2\mathrm{DS}_2$  VASc评分为1分（男）/2分（女）的患者需在考虑净临床获益的个体化特征及个人意愿的情况下，考虑应用OAC；（3） $\mathrm{CHA}_2\mathrm{DS}_2$  VASc评分为0分（男）/1分（女）的患者应避免抗凝治疗预防血栓栓塞。  $\mathrm{CHA}_2\mathrm{DS}_2$  VASc评分项中，年龄  $65\sim 74$  岁为1分、年龄  $\geq 75$  岁为2分，因此老年CCS合并房颤患者应依据上述(1)和(2)评分结果考虑是否及如何应用OAC。

2. 出血风险评估：目前主要采用HAS-BLED评分，来进行NVAF患者出血风险的评估[6,41]。HAS-BLED≤2分提示出血风险较低，≥3分提示出血风险较高，但出血风险高并不意味着必须停用抗凝治疗，而应该注意筛查并纠正增加出血风险的可逆因素，并在接受抗凝治疗后加强监测，比如严格控制高血压在达标范围以及监测INR并努力保证其在治疗窗的稳定性。

（三）老年CCS合并房颤患者的抗栓治疗方案

AFIRE研究显示，对于CCS合并NVAF患者，利伐沙班单药治疗疗效不劣于利伐沙班  $+1$  种抗血小板药物，而大出血发生率更低[42]；OAC-ALONE研究与AFIRE研究的荟萃分析显示，OAC单药治疗与  $\mathrm{OAC + SAPT}$  疗效相似，且具有更低的出血风险[43]。

推荐17：根据  $\mathrm{CHA}_2\mathrm{DS}_2\mathrm{VASc}$  评分，CCS合并房颤患者如具有抗凝指征，推荐应用卒中预防剂量的OAC单药治疗。

推荐18：对于高缺血风险、无高出血风险的CCS合并房颤患者可考虑在长期OAC基础上加用阿司匹林  $75\sim 100~\mathrm{mg / d}$  （或氯吡格雷  $75~\mathrm{mg / d})$  。

推荐19：对于适合非维生素K拮抗剂口服抗凝药物(NOAC)的患者，起始口服抗凝治疗时推荐NOAC优于维生素K拮抗剂(VKA)。

（四)CCS合并房颤患者PCI术后的抗栓治疗

荟萃分析显示，对于PCI术后患者，PCI治疗(OAC+SAPT)较三联抗栓治疗(OAC+DAPT)降低出血风险的效果更为显著，且对主要心血管不良事件发生率的影响相似[44-45]。出血风险可能与抗凝质量(INR的稳定性)呈负相关[46]；在大出血风险的影响因素中，出血危险因素可能大于联合抗栓方案本身[47]。考虑到CCS合并房颤患者需要长期OAC治疗，建议PCI术中选择新一代DES，以减少出血并发症[6]。

推荐20：对于高缺血/血栓风险且非高出血风险患者，PCI术后三联治疗(OAC+DAPT)1个月，双联治疗(OAC+SAPT)至12个月，之后长期维持OAC单药治疗。

推荐21：对于低缺血/血栓风险或高出血风险患者，PCI术后三联治疗(OAC+DAPT)至出院，之后应使用双联治疗(OAC+SAPT)6个月，之后长期OAC单药维持治疗。

推荐22：在OAC与抗血小板药物联合应用时，应首选NOAC。

推荐23：高龄、高血压、肝肾功能异常、卒中史等诸多因素与大出血有关，此类老年房颤患者PCI术后用药需严格评价栓塞及出血风险，有消化道出血风险的应注意质子泵抑制剂的使用。

# （五）经皮左心耳封堵术（LAAC）

LAAC是预防房颤患者血栓栓塞事件的策略之一。对于CHA2DS2-VASc评分  $\geq 2$  分（男）/  $\geq$  3分(女)但具有下列情况[48-50]：(1)不适合长期规范抗凝治疗；(2)长期规范抗凝治疗的基础上仍发生血栓栓塞事件；(3)HAS-BLED评分  $\geq 3$  ，或患者拒绝/不依从长期抗凝治疗等主客观因素限制了抗凝治疗者可考虑行LAAC。

推荐24：左心耳封堵术后常规推荐45d口服抗凝药治疗，45d确认封堵成功后，改为双联抗血小板治疗，6个月复查正常后更改为单药治疗。

推荐25：对于抗凝禁忌的患者，一种可行的方案是术后  $3\sim 6$  个月内双联抗血小板治疗，再转为长期单用阿司匹林。

推荐30：对于老年CCS合并VTE患者，应根据VTE发病时间及PCI时间综合制定抗栓治疗策略。

# （四）老年CCS择期PCI合并VTE

VTE患者考虑行PCI时应根据抗凝疗程是否完成及抗凝药物类型，采用不同的抗栓策略。

推荐39：如PCI无法延迟至抗凝结束，或需行紧急/急诊PCI，则应按PCI围术期进行药物管理，围术期后首选  $\mathrm{NOAC} + \mathrm{P2Y}_{12}$  受体拮抗剂，或华法林 $+\mathrm{P2Y}_{12}$  受体拮抗剂替代治疗。对于消化道出血的高危人群，合并应用胃肠黏膜保护剂，首选质子泵抑制剂(PPI)，若不耐受，改换为  $\mathbf{H}_2$  受体拮抗剂。如患者血栓风险高、出血风险低，可在PCI术后  $30\mathrm{d}$  内加用阿司匹林  $75\sim 100~\mathrm{mg / d}$  进行短期的三联治疗。

# 五、老年CCS抗栓治疗患者出血风险评估

# （一）出血风险评估工具介绍

老年CCS患者既存在高缺血风险，也存在高出血风险，抗栓治疗面临两难局面。这主要由老年患者的临床特点所决定，包括多病共存、复杂的冠脉病变、衰弱、跌倒、营养不良、肾功能受损等。老年CCS抗栓治疗患者出血风险评估应包括抗血小板治疗出血风险评估和抗凝治疗出血风险评估两部分。

目前临床常用的出血风险评估工具包括PRECISE-DAPT评分、HAS-BLED评分等。

ARC-HBR标准包括多个老年患者特点，可以在临床实践工作中尝试使用，作为PRECISE-DAPT评分和HAS-BLED评分的必要补充[58-59]。推荐应用该标准开展临床研究，进一步验证对老年CCS抗栓治疗患者出血风险预测价值[5,58-59]。

对老年CCS抗栓治疗患者进行出血风险评估前要进行老年综合评估，着重进行跌倒、多重用药、衰弱、痴呆等评估，根据评估结果，采取相应措施，进一步减少出血风险[60-69]，包括根据患者具体临床情况，积极采取措施纠正直立性低血压、改善脑供血、进行步态训练等，减少跌倒风险；减少药物不良反应叠加和药物相互作用，注意合用的抗血小板药物或抗凝药物，纠正低蛋白血症，减少多重用药出血风险；改善周围环境、教育照护者、选用单次给药不需剂量监测药物，增加认知功能受损的老年患者治疗安全性和依从性[70]。

老年CCS抗栓治疗患者出血风险评估还需要注意动态评估。老年患者各种临床情况和生理指标都不断变化（如肾功能、血压控制情况、贫血等），各种影响老年评估的因素不断波动，均会影响出血风险评估等级。临床工作者需严密观察老年患者临床情况，有计划进行动态监测和评估，随时掌握患者出血风险变化，及时优化抗栓治疗方案[71-75]。

推荐40：抗血小板治疗患者出血风险评估推荐采用PRECISE-DAPT评分。

推荐41：抗凝治疗患者出血风险评估推荐采用HAS-BLED评分，其预测精确度较高，且有较好颅内出血预测价值。

# （二）老年CCS抗栓治疗患者出血风险评估方法

老年CCS患者抗栓治疗出血风险评估，应充分考虑抗血小板治疗及抗凝治疗出血风险，结合老年综合评估，进行个体化风险评估。

推荐42：对于老年CCS不合并房颤或VTE患者，推荐PRECISE-DAPT评分进行抗栓治疗出血风险评估，结合冠脉病变复杂程度制定长期抗栓治疗策略(表10，图1)。

推荐45：抗栓治疗发生脑出血时，立即停药。

推荐46：华法林相关ICH：静脉滴注维生素K；可考虑将凝血酶原复合物（PCC)作为替代选择；如需可考虑输注血小板治疗。

推荐47：NOAC相关ICH：有条件者可用相应拮抗药物（如达比加群可使用依达赛珠单抗）；无特异性逆转剂可考虑PCC或活化凝血酶原复合物(aPCC)；如需要可考虑输注血小板治疗。

推荐48：普通肝素相关ICH：推荐使用硫酸鱼精蛋白治疗。

推荐49：抗血小板药物相关ICH：不推荐常规输注血小板治疗。

2. 胃肠道出血：衰老引起的胃肠道损伤增加了高龄患者抗栓治疗出血的风险，胃肠道是CCS

患者抗栓治疗并发出血最常见的部位[83-84]。质子泵抑制剂(PPI)可减少胃肠道出血风险，尤其是有胃肠道出血或溃疡史以及合并使用抗血小板治疗或OAC治疗的患者[6,26,85-88]。长期使用PPI可能会增加低镁血症风险，但监测血镁水平的作用尚不明确[2,6]。对于有明确适应证、需长期PPI治疗的患者而言，使用最小有效剂量是相对安全的，其获益远超风险。

推荐50：对幽门螺杆菌（HP）阳性患者应予HP根除治疗。

推荐51：阿司匹林、DAPT或OAC治疗的患者，如存在胃肠道出血高风险推荐同时使用PPI。

推荐52：部分PPI通过CYP2C19竞争性抑制氯吡格雷的抗血小板作用，可能影响其临床疗效，因此与氯吡格雷联用时，推荐使用受CYP2C19影响较小的PPI(如泮托拉唑、雷贝拉唑等)，不建议氯吡格雷与奥美拉唑或埃索美拉唑同时使用。

推荐53：一旦发生胃肠道出血，初诊  $24\mathrm{h}$  内应完成内镜检查；再出血风险高的患者应尽早使用大剂量PPI(80mg静脉滴注  $+8\mathrm{mg / h}$  静脉维持）持续治疗  $72\mathrm{h}$ ，并根据内镜下分型和止血结果及时调整。

推荐54：对内镜下止血失败，或存在复杂合并症/不宜外科手术的高龄患者，宜首先考虑行经导管动脉栓塞（TAE）。

推荐55：结肠镜检查是明确结直肠出血原因和部位的最重要手段，且可在内镜直视下进行止血治疗。

推荐56：胶囊内镜应视为上下消化道检查阴性、怀疑小肠出血患者的首选检查方式，建议择期检查的最佳时机为出血停止后  $72\mathrm{h}$ ，但不应超过2周。

3. 出血引起的贫血：抗栓治疗并发的出血可引起贫血，贫血是老年慢性冠心病患者出血性及缺血性事件风险的独立危险因素[89-90]。

推荐57：对于老年人，容量复苏问题需格外谨慎，推荐使用限制性容量复苏策略达到目标血压，直到出血控制。

推荐58：将低的初始血红蛋白水平视为与凝血变化相关的严重出血的指标，并重复检测血红蛋白。将血清乳酸和/或碱缺失作为估计和监测出血与休克程度的敏感性指标。

推荐59：对于有贫血症状或活动性出血患者的输血治疗指征建议为血红蛋白  $< 70\mathrm{g / L}$  ，推荐血

红蛋白的最大目标值不超过  $90\mathrm{g / L}$  ，输血适应证把握不当可能增高病死率。

推荐60：在大量输血期间监测离子钙水平并保持在正常范围内。建议使用氯化钙纠正低钙血症。

# （三）特殊人群的出血风险防治

1. 肾脏功能减低的出血风险的防治：肾小球滤过率(eGFR)下降与出血性卒中风险之间呈渐进、独立、反向的线性关系。慢性肾脏疾病(CKD)的患者以及在接受透析的终末期肾病(ESRD)患者更易发生胃肠道出血，且更为严重。对于肾功能不全的患者，2019年美国房颤管理指南提出，NOAC用药之前应评估肾功能，NVAF合并中重度CKD且  $\mathrm{CHA}_2\mathrm{DS}_2$  -VASc评分  $\geq 2$  分（男）/  $\geq 3$  分（女)的患者，根据血清肌酐的浓度水平选择不同类型的NOAC[91]。有回顾性研究表明，对于合并慢性肾脏病  $4\sim 5$  期或血液透析的NVAF患者，利伐沙班疗效与华法林相似，且其可显著降低大出血风险[92-93]。

推荐61：阿司匹林在CKD  $1\sim 5$  期均可使用且无需调整剂量，氯吡格雷和替格瑞洛在CKD  $1\sim 4$  期  $(\mathrm{eGFR}\geqslant 15\mathrm{ml}\cdot \mathrm{min}^{-1}\cdot 1.73\mathrm{m}^{-2})$  均可使用且无需调整剂量；在CKD5期患者中，氯吡格雷仅用于有选择性指征(如预防支架血栓)患者，不推荐使用替格瑞洛。

推荐62：在透析或肌酐清除率  $(\mathrm{CrCl}) < 15$  ml/min的患者抗凝治疗推荐首选华法林。

推荐63：在重度肾功能不全（CrCl  $15\sim 29~\mathrm{ml}/$  min)患者禁用达比加群酯，推荐口服Xa因子抑制剂应减量并慎用。

推荐64：中度肾功能不全（CrCl  $30\sim 49~\mathrm{ml}/$  min)患者使用DOAC时应减量，如利伐沙班15 mg,1次/d、艾多沙班  $30\mathrm{mg},1$  次/d或达比加群酯  $110\mathrm{mg},2$  次/d；轻中度肾功能不全（CrCl  $30\sim 95$  ml/min)或肾功能正常患者使用阿哌沙班时，满足下列3项中任意2项推荐使用低剂量方案(2.5  $\mathrm{mg},2$  次/d)：年龄  $\geqslant 80$  岁，体重  $\leqslant 60\mathrm{kg}$  ，肌酐  $\geqslant 133$ $\mu \mathrm{mol} / \mathrm{L}_{\circ}$

2. 肝功能不全的出血风险的防治：慢性肝病患者由于血栓和出血风险均增加，抗栓治疗难度增大[94-95]。目前尚缺乏肝病患者抗凝治疗的指南建议。由于肝功能受损，血浆白蛋白水平降低（血浆蛋白结合率：利伐沙班  $95\% >$  阿哌沙班  $85\% >$  艾多沙班  $55\% >$  达比加群  $35\%$  ）及细胞色素P450酶

活性降低（阿哌沙班和利伐沙班主要经CYP3A4代谢，艾多沙班少量经CYP3A4代谢，达比加群不经P450酶代谢），OACs在该类患者中的抗凝作用增强[27]。

推荐65：推荐采用Child-Pugh分级对肝受损程度进行分类。

推荐66：对于轻度肝功能损害的患者，阿司匹林、氯吡格雷、替格瑞洛等抗血小板药物均无需调整剂量。

推荐67：对于可能有出血倾向的中重度肝脏疾病患者，应谨慎使用氯吡格雷，禁用替格瑞洛。

推荐68：华法林可用于大多数肝病患者，是目前唯一用于严重肝功能不全患者的OAC。凝血监测INR，范围是  $2.0\sim 3.0$  ，但鉴于华法林血药浓度的不稳定性，且临床往往难以区分INR值升高是由于华法林还是基础疾病进展所致，因此，华法林用于肝功能不全患者的安全性仍有待进一步确认。

推荐69：对于轻度肝功能损害患者，使用NOAC均无需调整剂量；中度肝功能损害患者可慎用常规剂量达比加群、阿哌沙班或艾多沙班，不推荐利伐沙班。

3. 高龄或衰弱患者的出血风险的防治：高龄（≥75岁）或衰弱是出血的主要危险因素之一。从临床研究证据来看，大多数抗血小板研究人群为中位年龄<65岁患者，而65岁以上老年患者数据仍然缺乏，尤其高龄患者常被排除在随机对照研究之外，因此高龄患者使用抗血小板治疗时应谨慎[96-98]。

高龄患者多病共患、多重用药现象普遍存在，尤以华法林的药物相互作用涉及范围最广，而NOAC在与抗真菌药、免疫抑制剂、决奈达隆等某些特定药物合用时也存在明确的相互作用，会增加出血风险，需格外小心[27]。应用HAS-BLED评分可识别出血高危患者（评分  $\geq 3$  分为高危），对可控制的出血危险因素进行纠正，并在整个抗凝治疗中定期随诊[41]。

推荐70：对于高龄患者出血规避策略应着重关注抗栓药物优化，并尽量规避PCI及外科手术治疗。

推荐71：接受手术的高龄患者应尽量避免联合应用抗血小板和抗凝治疗，如此类患者必须手术，则尽量推迟，直至抗栓疗程终止或可减量时，并于术前充分评估出血风险。

推荐72：对于可疑血小板功能不全的高龄患

者, 建议使用床旁血小板功能检测。平均血小板计数  $< 50 \times 10^{9} / \mathrm{L}$  是应用抗血小板药物和抗凝治疗的临界点。必要时可予以输注单采血小板以维持血小板计数  $\geqslant 50 \times 10^{9} / \mathrm{L}$  (凝血因子不足时也可考虑输注冷沉淀维持纤维蛋白原  $>100 \mathrm{mg} / \mathrm{dl}$  )。

老年CCS高危患者在临床治疗选择时应谨慎权衡抗栓药物的有效性和安全性。对于老年CCS高危患者，应准确评估其出血及缺血风险，掌握DPI及DAPT强化抗栓的适应证，定期随访并及时纠正缺血或出血的风险因素，准确判定出血程度和部位并及时应对处理，使患者抗栓获益最大化、出血风险最小化。
