# 引言

腺嘌呤核苷简称腺苷，是一种在体内广泛分布的内源性核苷，在人类生理学中具有重要功能[1]。在氧含量正常时，腺苷保持低水平持续释放；而当组织器官受到缺血、缺氧、外伤及炎症等刺激时，腺苷的浓度会反应性升高，并可通过调节局部组织和细胞的能量代谢，减轻组织损伤、促进血管新生和组织恢复等[2]。腺苷受体在体内广泛表达，存在于机体大多数组织细胞表面，目前已发现4种腺苷受体，即腺苷A1、A2A、A2B和A3受体。腺苷与上述受体结合后，可发挥不同的药理学作用，调节心血管系统等功能[3]。

目前，腺苷已在多种疾病的诊疗中广泛应用，如无创性心血管影像学诊断（负荷核素心肌灌注显像、超声心动图等）、治疗室上性心动过速、防治经皮冠状动脉介入治疗(percutaneous coronary intervention, PCI)术中无复流或慢血流等，临床证据日益增加并获得多部国内外权威指南、共识等的高度推荐。例如，《冠状动脉微血管疾病诊断和治疗中国专家共识(2023版)》[4]明确指出，腺苷是最常用的检测冠状动脉微血管功能的非内皮依赖性舒张血管的药物；《2019年欧洲心脏病学会室上性心动过速患者管理指南》[5]提出，对血流动力学稳定的刺激迷走神经无效的房室结折返性心动过速患者，腺苷为I类推荐药物；而《急性ST段抬高型心肌梗死诊断和治疗指南(2019)》[6]则指出，在预防或减轻慢血流或无复流方面，腺苷为IIa类推荐药物；此外，2019年《英国超声心动图学会实践指南：冠心病负荷超声心动图检查》[7]提出，腺苷是负荷超声心动图检查常用的负荷药物；《核素心肌显像临床应用指南(2018)》[8]则推荐，负荷核素心肌灌注显像的药物负荷试验中的负荷药物可以选择腺苷。值得注意的是，早期国内腺苷可及性较差，尚无充足的药物供临床应用，常常存在三磷酸腺苷(adenosine triphosphate, ATP)替代其应用的情况。然而，ATP是腺苷的前体，二者虽然有所关联，但有显著的区别。例如，腺苷的分子式为  $\mathrm{C_{10}H_{13}N_5O_4}$ ，而ATP的分子式为 $\mathrm{C_{10}H_{16}N_5O_{13}P_3}$ ，二者化学结构不同。此外，ATP为体内能量的主要来源，参与脂肪、蛋白质以及糖等的代谢，在临床上如三磷酸腺苷二钠注射液主要用于进行性肌萎缩、脑出血后遗症、心功能不全等的辅助治疗，尚无治疗阵发性室上性心动过速及冠心病诊断等相关适应证。当前，腺苷已在国内广泛可及，在临床相关诊疗中无需再使用ATP或其他药物替代应用。为进一步帮助临床医师、药师全面了解腺苷，并正确、规范地使用腺苷，国内心血管病、影像学及药学领域的专家共同制订了《腺苷注射液在心血管疾病诊疗中的临床应用中国专家共识》。

本共识系统检索了PubMed、Cochrane Library、中国生物医学期刊文献数据库、万方数据库及中国知网等数据库，检索时间为建库至2024年5月31日，检索词包含“腺苷”“Adenosine”等，检出文献包括基础药理学研究、临床试验、临床实践指南共识及相关综述等，并筛选其中66篇文献进行重点研究。经国内心血管病、影像学及药学相关领域专家共同沟通讨论后，根据现有文献证据及临床实践经验形成推荐意见。

# 二、药物代谢动力学

腺苷可在全身各处组织器官产生，血管内皮细胞、平滑肌细胞和心肌细胞等均能迅速地合成和转运腺苷，并调节细胞外液如血浆中的腺苷浓度。在来源上，腺苷可由S-腺苷同型半胱氨酸水解及单磷酸腺苷经  $5^{\prime}$  -核苷酸酶脱磷酸产生。

外源性腺苷主要通过红细胞和血管内皮细胞摄取而在机体循环中分布。腺苷不经肝肾代谢，血中半衰期极短（不超过10s），其代谢途径包括：（1）腺苷与腺苷磷酸酶亲和力大于腺苷与脱氨酶的亲和力，故大部分腺苷再次磷酸化生成单磷酸腺苷，完成腺苷再循环；(2)腺苷在脱氨酶作用下，生成次

黄嘌呤核苷酸和次黄嘌呤，最后生成终末代谢产物尿酸[16-17]。细胞外腺苷主要通过细胞摄取而从血浆中清除，其余部分可通过腺苷脱氨形式进行脱氨。

# 腺苷在心血管疾病诊断领域中的临床应用

# 一、无创性心血管影像学诊断

（一）腺苷负荷超声心动图在冠状动脉大血管及微血管功能障碍诊断中的应用

腺苷负荷超声心动图在冠心病的诊断中有着良好的临床应用价值，敏感度、特异度和准确性较高，在国内外得到广泛应用。Martin等[18]的研究表明，腺苷负荷超声心动图比多巴酚丁胺或双嘧达莫负荷超声心动图诊断冠心病的特异度更高，且较少引起患者不良反应，安全性更好。三维斑点追踪技术结合腺苷负荷超声心动图可定量评价心肌收缩功能的微小改变，能较好地识别心肌缺血，对定位诊断严重冠状动脉病变的准确性也较高[19]。

心肌声学造影又称心肌造影超声心动图，可评估心肌灌注、存活心肌的收缩功能及血流储备情况等。通过观察腺苷负荷心肌造影超声心动图前后室壁运动及心肌灌注变化可识别冠心病心肌缺血，估测冠状动脉狭窄程度，对稳定型心绞痛的诊断价值较高[20]。例如在高血压患者中，腺苷负荷心肌造影超声心动图对冠心病的总体诊断准确性与核素心肌灌注显像相似，且在左回旋支区域更具有

# （二）腺苷负荷核素心肌灌注显像在冠状动脉大血管及微血管功能障碍诊断中的应用

核素心肌灌注显像（myocardial perfusion imaging, MPI)是一种无创、有效评价冠心病心肌缺血的功能性检查方法，准确性高、循证医学证据充分，可直接反映冠状动脉供血心肌的血流灌注状态，明确心肌缺血部位、范围及程度。根据显像设备的不同，心肌灌注显像可分为单光子发射型计算机断层显像（single photon emission computed tomography, SPECT)和正电子发射型计算机断层显像（positron emission tomography, PET）。在负荷状态下，正常和病变的冠状动脉血流量和心肌血流灌注显像剂的相对分布出现差异，提高了病变的检出率，还可以反映狭窄冠状动脉血流储备功能[25]。总体上，药物负荷SPECT心肌显像诊断冠心病的敏感度为  $88\% \sim 91\%$  ，特异度为  $75\% \sim 90\%$  [26]。李冬香等[27]的研究显示，腺苷负荷核素MPI诊断冠心病的敏感度为  $92\%$  ，特异度为  $91\%$  ，诊断准确性为  $94\%$  。与其他无创诊断技术相比，一项对70例冠心病心肌缺血患者的研究发现，腺苷负荷核素MPI对冠心病心肌缺血的诊断符合率较高，且其联合心电图对不同病变部位的诊断准确性较好，具有较高的临床应用价值[28]。

核素MPI为有效诊断冠状动脉微血管疾病的影像学检查。SPECT利用201铊或  $99\mathrm{m}$  锡标记的示踪剂，记录静息和腺苷等药物负荷状态下心肌中的放射活性，进而可发现两种状态下的节段性心肌灌注减低、灌注缺损或灌注再分布征象，在心外膜下的冠状动脉无明显狭窄的情况下，有助于诊断冠状动

脉微血管疾病所致的心肌缺血，诊断敏感度和阴性预测价值较高。而PET则基于使用标记有发射正电子的同位素的示踪剂，具有更高分辨率，可实现示踪剂动力学的快速动态成像[29]，并可反映冠状动脉微血管功能障碍[30]。现有研究已证实，腺苷结合PET可以可靠地检查冠状动脉微血管功能障碍[31-32]。

在安全性方面，相较其他血管扩张剂如ATP，腺苷安全性更高。国内一项研究发现[33]，应用负荷核素MPI测定CFR时，相较ATP负荷，腺苷负荷核素MPI的心悸、头痛及气促等不良反应发生率显著降低，安全性更高。此外，既往研究发现，腺苷的不良反应大多数表现为血压轻度下降、头晕、气短和面部潮红等，且大多数不良反应在检查结束后很快自行缓解，目前还尚未发现有患者出现房室传导阻滞等严重不良反应，因此腺苷负荷核素MPI安全性较好[34]。

# （三）腺苷负荷心脏磁共振在冠状动脉大血管及微血管功能障碍诊断中的应用

近年来，心脏磁共振（cardiac magnetic resonance, CMR）在冠状动脉成像和评估冠状动脉及心肌功能等应用方面越来越广泛。腺苷负荷CMR可评估心脏功能、缺血情况、梗死瘢痕区的存在及大小、冠状动脉异常情况及微血管阻塞疾病的预后。多项研究提示，以冠状动脉血流储备分数结果为金标准，负荷磁共振心肌灌注成像的敏感度为 $84\% \sim 90\%$  ，特异度为  $89\% \sim 92\%$  [35]。腺苷负荷的CMR心肌灌注成像，对于已知或可疑冠心病的患者具有较好的诊断价值[36]

血管扩张剂（如腺苷）负荷的CMR可通过心肌与血池信号对比或者注射顺磁性造影剂引起的信号强度改变来评价冠心病确诊或疑诊患者的心肌缺血和微血管阻塞状况。研究发现，腺苷负荷CMR在患有冠状动脉微血管功能障碍和潜在结构性心脏病（如扩张性心肌病）患者中的诊断和预后评估方面具有较高价值[37]。CMR评估冠状动脉微血管功能障碍具有独特优势，如空间分辨率较高和无离子辐射危险等。此外，与TTDE和有创技术相比，CMR还可以同时评估所有冠状动脉区域，并且图像质量与患者特征（如肥胖）无关[38]。

腺苷负荷CMR的安全性较高。一项评估其安全性的研究显示，仅少数患者出现轻微症状，包括

潮红、胸部不适等，仅1例患者出现一过性高度房室传导阻滞，无严重并发症（如死亡、过敏性休克、室性心动过速等），证实腺苷负荷CMR安全可行[39]。

（四）腺苷负荷CT心肌灌注成像在冠状动脉大血管及微血管功能障碍诊断中的应用

近年来，CT技术快速发展，其空间和时间分辨率均得到大幅提高、扫描时间明显缩短等推动了心肌灌注成像的发展，随着CT心肌灌注成像的发展及成熟，其逐步应用于心脏疾病尤其是缺血性心脏病的诊断[40]，可准确地反映冠状动脉狭窄的血流动力学意义，为临床决策提供准确依据。

目前CT心肌灌注扫描方案通常分为3个阶段，即冠状动脉CT血管成像（computed tomography angiography，CTA)扫描、药物负荷CT心肌灌注成像扫描和碘延迟强化扫描。CT心肌灌注成像结合冠状动脉CTA的“一站式”检查具有独特优势，可同时从解剖学和功能学特征两方面对冠状动脉及其支配心肌进行综合评估[41]，较单独冠状动脉CTA检查可提高冠心病诊断的特异性[42-43]。腺苷为目前负荷CT心肌灌注成像扫描的常用负荷药物，不良反应轻微，安全性较高。在腺苷的作用下，正常血管可显著扩张，与病变血管的扩张程度不同，故通过最大化的血管扩张差异区分血管；扩张差异导致冠状动脉血流量不同，出现心肌的灌注差异，表现为病变血管供血区心肌低密度影。研究发现，腺苷负荷的CT心肌灌注成像结合冠状动脉CTA诊断冠状动脉狭窄的敏感度、特异度及准确性分别为 $84\%$ 、 $89\%$ 和 $88\%$ [44]。同样，Ko等[45]发现，腺苷负荷的CT心肌灌注成像可良好识别缺血区域，特异度大于 $80\%$ 。

负荷CT心肌灌注成像可以获得整个心脏功能学信息，所得相关定量参数在一定程度上能够详细反映病症问题，同时也可反映微循环功能障碍。一项研究发现，腺苷负荷的CT心肌灌注成像可以良好评估心肌灌注，并提供有关冠状动脉微血管功能的信息[46]，而冠状动脉CTA与CT心肌灌注成像的结合应用，同样也可对明确识别微循环病变提供更加准确的指导[47]，进而为冠心病的精准诊疗提供更加充分及有效的证据。

术测量冠状动脉血流储备的临床价值

冠状动脉内多普勒血流导丝技术应用多普勒血流速度描记仪及其配套的多普勒导丝，经动脉插管至冠状动脉远端，记录血流频谱，然后注射腺苷扩张血管以达到最大充血反应，测量充血状态下的冠状动脉血流速度。计算充血状态和基础状态的舒张期冠状动脉血流速度即可得出CFR。多普勒导丝技术在冠状动脉介入领域中具有重要作用。Mazur等[48]的研究发现，心肌梗死患者在经皮冠状动脉腔内成形术前应用冠状动脉内多普勒导丝（应用腺苷诱发充血)测量梗死相关冠状动脉的CFR有助于决策后续血运重建治疗。而另一项研究则表明，在应用腺苷并通过冠状动脉内多普勒导丝技术测量冠状动脉血流速度储备(coronary flow velocity reserve, CFVR)后，对冠状动脉明显狭窄但CFVR大于或等于2.0的患者延迟进行经皮冠状动脉腔内成形术是安全的[49]，并可用于辅助评估介入治疗的必要性。

（二）腺苷结合冠状动脉血流储备分数检测评估罪犯血管、指导血运重建的临床价值

冠状动脉血流储备分数（fractional flow reserve, FFR)是评估冠状动脉血流的功能学和生理学指标，定义为存在狭窄病变情况下该冠状动脉

提供给心肌的最大血流量与理论上无狭窄情况下心肌所能获得最大血流量的比值。

FFR概念的基础是假设在微循环血管最大充血状态下，灌注压的变化可反映血流量的变化，因此使用血管扩张剂确保诱发微循环最大程度充血，对FFR的准确测量至关重要。其中，腺苷可以使心肌达到最大充血状态，让微循环阻力降到最低且保持不变，与其他血管扩张剂如硝普钠作用相当[50]。研究表明，与冠状动脉造影指导的血运重建相比，腺苷负荷FFR在不增加冠状动脉旁路移植术(coronaryartery bypass grafting,CABG)术后并发症的同时减少了CABG手术次数[51]，即避免了部分不必要的血运重建治疗。此外研究发现，在接受梗死相关动脉直接PCI的多支血管病变的ST段抬高型心肌梗死（ST-segmentelevationmyocardialinfarction,STEMI)患者中，腺苷负荷FFR指导的非梗死相关动脉完全血运重建可使复合终点(全因死亡、非致死性心肌梗死、任何血运重建和脑血管事件)风险显著降低[52]。而FRAME-AMI研究结果表明，在合并多支血管病变的急性心肌梗死患者中，根据腺苷负荷FFR检测结果而有目的地选择非梗死相关动脉病变进行PCI治疗，可显著降低死亡、心肌梗死或再次血运重建的风险，优于基于血管造影指导的常规PCI治疗策略[53]。因此，腺苷负荷FFR可用于指导临床治疗策略的选择及判断预后。

# 推荐意见

- 在应用冠状动脉内多普勒血流导丝技术测定冠状动脉血流储备或进行血流储备分数检查时，可选择腺苷扩张血管以达到最大充血反应，进而准确评估冠状动脉功能学情况。推荐应用腺苷  $140\sim 180~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  静脉输注，持续输注  $2\sim 3\mathrm{min}$

# 三、在心房颤动射频消融术中确认隐匿性传导（一）心房颤动射频消融术的临床应用现状

心房颤动（以下简称房颤）是最常见的心律失常，可导致心力衰竭和卒中，是房颤致残和死亡的主要原因[54]。在药物治疗受限或控制不佳时，导管消融治疗是一种选择。目前临床指南等均建议将肺静脉电隔离作为难治性症状性房颤治疗的I类适应证[55]。作为房颤消融的基石性治疗策略，肺静

脉电隔离可使患者在消融术后相当长一段时间维持窦性心律，但术后仍易复发。研究表明，单次消融1年成功率仅为  $50\% \sim 75\%$  [56-58]。房颤消融术后房颤复发的主要原因在于肺静脉与心房之间隐匿性传导的恢复。然而，房颤肺静脉电隔离后应用腺苷可以使隐匿性传导得以显现，对隐匿性传导实施进一步消融后有助于提高房颤消融远期成功率。

# （二）腺苷在心房颤动射频消融术中的应用

腺苷能有效诱发肺静脉电位隐匿性传导，指导房颤消融，提高消融成功率。ADVICE研究[59]证实了腺苷在阵发性房颤射频消融术中的应用价值，该研究纳入534例患者，腺苷暴露了284例患者的隐匿性肺静脉传导。对发生隐匿性传导的患者进行随机分组，一组接受腺苷指导进一步消融以消除隐匿传导，而另一组未接受腺苷指导，随访1年。结果发现，147例接受腺苷指导消融的患者中有102例未发生症状性房性快速性心律失常，而在137例未接受进一步消融的患者中仅有58例未发生上述心律失常，复发绝对风险显著降低  $27.1\%$  。另一项研究发现[60]，阵发性房颤患者在接受肺静脉隔离后行腺苷试验并进一步消融后，未行腺苷试验组的房颤复发率显著高于行腺苷试验组，证实在肺静脉隔离后进行腺苷试验确认有无隐匿性传导，将有助于改善阵发性房颤患者导管消融术后的预后。其他2项包含多个研究的meta分析同样表明[61-62]，在肺静脉隔离后，早期、足量应用腺苷并对诱发的再连接进行消融，可能对提高房颤患者消融成功率有益。

# 推荐意见

- 初始肺静脉隔离  $20 \mathrm{~min}$  后, 应用腺苷确认肺静脉电位隐匿性传导。常规静脉使用腺苷  $(12 \sim 18 \mathrm{mg})$  验证消融终点, 如发现肺静脉-心房电传导恢复, 则应进行补点消融, 以提高肺静脉隔离成功率, 减少后续房颤复发。

室上性心动过速广义上指一组起源于房室交界区及其以上传导系统的心动过速，包括窦性心动过速、阵发性室上性心动过速及房性心动过速等；狭义上特指阵发性室上性心动过速，是最常见的室上性心律失常。阵发性室上性心动过速的发病率

约每年  $0.36 \text{‰}$  [63]，可见于多年龄段，多发于女性和老年人。根据阵发性室上性心动过速发生的电生理机制，可将其分为房室结折返性心动过速（atrioventricular node reentrant tachycardia, AVNRT）及房室折返性心动过速（atrioventricular reentrant tachycardia, AVRT)2种类型，其中AVNRT最常见，占比近2/3，女性占比高于男性；而AVRT则相反[64]。

# 二、经皮冠状动脉介入治疗围手术期心肌保护（一)PCI引起的无复流/慢血流与再灌注损伤

1. 无复流/慢血流：急性心肌梗死治疗的关键是尽早给予冠状动脉血运重建，包括药物溶栓、PCI及CABG等。然而在临床实践中，存在某些心外膜血管在再灌注治疗期间已解除阻塞，但相应心肌组织并没有完全有效恢复血流灌注的情况，即无复流现象[72]，其发生的主要原因为微血管内皮细胞肿胀、微血管外间质中由渗出液引起的组织间压力增高以及微血管堵塞等，发生率为  $5\% \sim 50\%$  ，可使住院死亡和心肌梗死发生率增加  $5\sim 10$  倍[73-74]。无复流本身虽不是心肌梗死的最初原因，但其是继续缺血、梗死延展、心室重构及心肌梗死急性期高死亡率的预测指标，也是冠状动脉微循环严重损伤的标志。除无复流外，慢血流亦为PCI的常见并发症，同样意味着微循环水平血流灌注仍未完全恢复[75]。在发病机制上，其可能与内皮功能障碍、炎症、氧化应激及微血管远端栓塞等多种因素有关，可诱发急性冠状动脉综合征、恶性室性心律失常，甚至猝死[76-77]。因此，PCI不仅要解除心外膜下冠状动脉的阻塞，同时还要注意保护冠状动脉微血管功能，降低PCI术中慢血流或无复流的发生率，以有效挽救心肌细胞，改善患者的临床预后。

2.再灌注损伤：PCI等再灌注治疗诱发的心肌损伤一直是困扰心脏介入医师的一项难题[78]，具体指冠状动脉闭塞后，发生于血管再通、心肌再灌注恢复时的细胞损伤和功能障碍，包括致死性或不可

逆的损伤如梗死带延展和可逆的损伤如心肌顿抑及微血管内皮损伤等，与氧自由基损害、钙离子超载，以及与白细胞激活、趋化黏附相关的炎性损害等有关[79]。PCI术中的再灌注损伤会导致心肌发生形态和功能的异常变化，甚至可能使部分缺血存活的心肌坏死而失去活性，最终导致心肌损害。

# （二）腺苷改善PCI引起的无复流/慢血流与再灌注损伤

1. 防治无复流/慢血流：为提高心肌灌注水平并防治慢血流或无复流的发生，很多药物已经应用于临床，包括腺苷、维拉帕米、硝普钠等。其中，腺苷除了可以扩张血管，提高心肌灌注水平外，还能减少缺血区中性粒细胞数，保持缺血区内皮相对完整及减少氧自由基的生成[80-81]，已被证实且推荐用于改善微循环障碍，防治慢血流或无复流[82]。研究发现，腺苷可有效改善微循环障碍，并可降低患者 $24\mathrm{h}$  内肌酸激酶同工酶（creatinekinase-MB,CK-MB)峰值，减轻心肌再灌注损伤[83]。另一项纳入240例STEMI患者的随机对照研究发现，在冠状动脉内应用腺苷可改善心肌血流灌注，减少微循环阻塞[84]。此外，有研究同样表明，无论以 $50~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$ 还是  $70~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  静脉输注腺苷  $3\mathrm{h}$ ，均可显著减少无复流的发生[85]。而在复杂PCI(如旋磨)手术中，预防性冠状动脉内注入腺苷也被证明可减少无复流[86]。多项研究表明，腺苷应用安全且患者耐受性良好[80,87]。

2.减轻再灌注损伤，减小心肌梗死面积：腺苷目前已被证实为一种安全、有效的心肌保护药物，可减轻心肌再灌注损伤并缩小心肌梗死面积，改善患者预后。自Ely等[88]研究发现腺苷对心肌细胞存在直接保护作用以来，对腺苷心肌保护作用的研究便不断深入并取得了重要进展。腺苷可通过激动A1、A2A、A2B及A34种腺苷受体，发挥对心肌的多靶点综合保护效应，主要表现为以下方面：（1)触发和介导心肌缺血预适应。缺血预适应指当心肌经历短时间的可逆性缺血过程之后，对于继发的长期持续性缺血可产生较强的耐受力。在早期阶段，内源性腺苷是触发因子之一[89-90]。一过性的外源性腺苷可模拟出拮抗梗死的早期阶段，而在一过性缺血或长时间持续缺血刺激时阻断腺苷受体将减弱心肌保护作用。(2)负性变力、变时和变传导作用。腺苷可降低心脏对儿茶酚胺的反应性，减少心

肌耗氧。(3)扩张冠状动脉及微循环，增加心肌供氧并抑制血小板聚集。(4)减少氧自由基产生，抑制 $\mathrm{Ca^{2+}}$  超载和中性粒细胞介导的炎性反应[10]。总之，腺苷可通过上述多种机制改善心肌灌注，减轻再灌注损伤，保护心肌。

AMISTAD研究[91]将236例发病  $6\mathrm{h}$  内的急性STEMI患者随机分为腺苷组和安慰剂组，并接受溶栓治疗。其中腺苷组经静脉以  $70~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  持续注射腺苷  $3\mathrm{h}$ ，安慰剂组给予生理盐水。结果显示，腺苷组前壁心肌梗死面积相对缩小  $67\%$  ，心肌挽救指数显著增加。AMISTAD-Ⅱ研究则再次验证腺苷在挽救濒死心肌，缩小心肌梗死面积中的价值。其纳入2118例前壁STEMI患者，随机分为安慰剂组、 $50~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  腺苷组及  $70~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  腺苷组，均在溶栓开始或冠状动脉介入治疗前 $15\mathrm{min}$  内应用腺苷或安慰剂并持续静脉注射  $3\mathrm{h}$  。结果显示，应用  $70~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  的腺苷可明显减少 $57\%$  的心肌梗死面积[92]。Zhang等[85]的研究则发现腺苷可使STEMI患者术后CK-MB的峰值显著降低。此外，meta分析结果也表明，腺苷治疗后可使STEMI患者心力衰竭以及短期和长期主要不良心血管事件发生风险显著降低[93-94]。

# 推荐意见

- 冠状动脉内注射腺苷  $100 \sim 200 \mu \mathrm{g} /$  次 (一般左冠状动脉给药建议不超过  $200 \mu \mathrm{g} /$  次, 右冠状动脉不超过  $100 \mu \mathrm{g} /$  次, 均可多次重复给药), 有助于预防或减轻急诊 PCI 术中的慢血流或无复流。  
·在急诊PCI术前  $15\mathrm{min}$  内以  $70~\mu \mathrm{g}\cdot \mathrm{kg}^{-1}\cdot \mathrm{min}^{-1}$  持续静脉输注腺苷  $3\mathrm{h}$  ，可减轻再灌注损伤，减小心肌梗死面积，保护心肌。

# 三、体外循环心脏停搏下的心脏手术

据《2022年中国心血管外科手术和体外循环数据白皮书》所示[95]，2022年726家医院共开展心脏大血管外科手术量为263292例，其中体外循环下手术总量共计159949例，占总手术量的  $60.7\%$

当前，高钾停搏液在临床应用较广泛，发挥了一定的心肌保护作用。然而，由于体外循环心脏停搏手术日趋复杂等原因，传统高钾停搏液的心肌保护作用效果已较难满足临床需求。在体外循环心脏手术的临床应用中，腺苷可以提升心肌对缺血再

灌注损伤的耐受性。临床发现，腺苷可以作为良好的心脏停搏液添加剂，诱导心脏快速停搏，并增强停搏液的心肌保护效果。目前认为，腺苷应用于心脏手术中的心肌保护作用主要体现在[96-97]：(1)增加心脏自主复跳率；(2)缩短心脏停搏诱导时间；(3)促进再灌注时心肌功能的恢复，增强心脏保护效果等。

作为经典的心肌保护策略，腺苷在体外循环心脏停搏下心脏手术中的临床用法主要为在心脏停搏液中直接添加  $1\sim 2\mathrm{mmol / L}$  的腺苷以增强常规心脏停搏液的心肌保护效果，可表现为心肌损伤的血浆标志物水平降低和/或伴有明显的临床获益等。一项研究发现[98]，在择期CABG患者中，含  $2\mathrm{mmol / L}$  腺苷的高钾冷血停搏液可使心肌梗死发生率显著降低，不良事件发生率明显减少。另一项研究将60例择期CABG患者随机分为两组，分别使用高钾停搏液和  $1.2\mathrm{mmol / L}$  腺苷替代高钾的停搏液。结果显示，腺苷可使停搏诱导时间显著缩短，术后新发房颤发生率也明显降低[99]。此外，腺苷在婴幼儿体外循环心脏手术中同样表现出良好的心肌保护作用。邢家林等[100]的研究发现，行体外循环下心内直视术的先天性心脏病患儿在应用含  $1\mathrm{mmol / L}$  腺苷的心脏停搏液后，心脏停搏耗时更短，重症加强护理病房  $1\textrm{h}$  时的血浆心肌肌钙蛋白IocardiactroponinI,cTnI)含量明显降低。其他研究显示，在停搏液中添加  $1\sim 2\mathrm{mmol / L}$  的腺苷可以在加快心脏停搏的同时减轻术中心肌缺血和缺氧，提供心肌保护作用，显著降低心肌肌钙蛋白T-cardiactroponinT,  $\mathrm{cTnT})$  、cTnI、白细胞介素-6(interleukin-6,IL-6)及白细胞介素-8(interleukin-8,IL-8)等水平[101-102]

腺苷应用安全性总体较高，但其可能会带来低血压或负性频率等不良反应，因此使用时应该注意监测并随时进行干预。恶性不良事件未见报道。

# 推荐意见

- 腺苷作为体外循环心脏停搏手术中常规心脏停搏液的添加剂（可在停搏液中直接加入腺苷，终浓度为  $1\sim 2\mathrm{mmol / L}$ ），其能够诱导心脏加快停搏，减少术后不良心血管事件发生，增强常规停搏液的心肌保护作用，整体应用安全、有效。

# 腺苷临床应用的注意事项

腺苷可经静脉或冠状动脉内注射给药；渗透压摩尔浓度比为  $0.9\sim 1.1$

# 一、主要不良反应及处理

腺苷较常见的不良反应为面部潮红、呼吸困难、支气管痉挛、胸部紧压感以及恶心和头晕等。较罕见的不良反应有出汗、心悸、过度换气、头部压迫感、焦虑、视力模糊、心动过缓、心脏停搏和头痛等，以上不良反应为轻度，持续时间短（常短于  $1\mathrm{min}$  且患者通常能较好耐受，基本停药后可自行恢复。

如冠状动脉注射腺苷后出现停搏，停药未立刻恢复，可让患者轻轻咳嗽几声即可恢复；如静脉输注腺苷后出现胸闷、胸痛或心动过缓，则减量使用或停用，具体根据不良反应的严重程度决定处理策略。

腺苷在血中半衰期小于  $10 \mathrm{~s}$ , 其不良反应在停药后很快消除, 但延迟和持续的药效仍可观察到。外源性腺苷既不在肾脏清除, 也不在肝脏降解, 故腺苷的作用不受肝肾功能的影响。腺苷受体拮抗剂, 例如氨茶碱, 可用于终止持续性的不良反应 (5~ $15 \mathrm{mg}$  稀释成  $20 \mathrm{ml}$  于  $1 \sim 2 \mathrm{~min}$  内注入或  $250 \mathrm{mg}$  稀释成  $20 \mathrm{ml}$  于  $5 \mathrm{~min}$  内注入)。若氨茶碱不能消除缺血反应, 应使用硝酸酯类药物。

# 二、禁忌及慎用人群

Ⅱ度及以上房室传导阻滞（置入心脏起搏器者除外）患者、病态窦房结综合征（置入心脏起搏器者除外）患者、已知或估计有支气管狭窄或支气管痉挛的肺部疾病患者（如哮喘）、对腺苷类药物过敏者禁用。

I度房室传导阻滞及房室束传导分支阻滞、植物性神经功能障碍、瓣膜狭窄性心脏病、心包炎或心包积液、狭窄性颈动脉疾病、未校正的血容量减少以及与支气管狭窄无关的阻塞性肺部疾病的患者(如肺气肿)应慎用腺苷。

# 三、药物相互作用

腺苷的作用可被腺苷受体拮抗剂（如咖啡因、氨茶碱等）抑制，而被核苷转运抑制剂（如双嘧达莫）增强；可能抑制或增强腺苷作用的药物应在使用腺苷前至少停药5个半衰期。腺苷与其他心血管活性药物（如β受体阻滞剂和强心苷类药物等）联用时，未见明显不良相互作用。然而，由于可能对窦房结或房室结产生相加或协同抑制作用，在合

并上述药物时应慎用腺苷。

# 总结和展望

腺苷通过激动腺苷A1、A2A、A2B和A3受体发挥多靶点综合作用，临床可用于终止室上性心动过速、防治PCI术中无复流或慢血流、PCI围手术期心肌保护、体外循环心脏停搏液添加剂及作为诊断负荷试验药物，如负荷超声心动图、负荷核素心肌灌注显像、血流储备分数检测，以及房颤射频消融终点的判定等，在心血管疾病诊断和治疗中发挥着重要的作用。此外，腺苷起效快，半衰期短，不经肝肾代谢，不良反应轻微且常呈一过性，临床应用安全性高。未来研究应聚焦开发腺苷受体调节剂，以根据不同病理状态减弱或放大腺苷反应，期望腺苷能够在临床相关疾病的诊疗中更科学合理地应用。
