# 慢病随访共识（Docker smoke 语料）

## 随访信息

本文件是受控的部署验证资料，不替代真实临床指南。它用于确认空卷启动后，服务能够在
同一 Compose 网络中访问 embedding Provider、Qdrant、PostgreSQL 与 Redis，并把解析后
的 Markdown 内容写入新的向量集合。完整知识库的离线重建应在专门的运维任务中执行，
并记录数据版本、耗时、失败文件和质量评估结果。

## 重启保持

索引完成后重启 API 不应丢失 RAG readiness；健康检查需要重新确认服务能访问已有的
持久卷和索引集合。这个行为验证部署基础设施，不代表任何个体的诊断、风险分层或治疗
推荐已经通过临床验收。
