# 老年综合评估指南（Docker smoke 语料）

## 综合评估范围

该文档仅用于验证 Docker 空卷环境中的 Markdown 解析、分块、embedding、向量写入和
readiness 链路，不作为临床建议或真实知识库内容。老年综合评估通常覆盖基础疾病、功能
状态、认知与情绪、营养、跌倒风险、社会支持和照护目标。系统应保留来源、章节和版本
等可追溯信息，以便在检索结果中展示其依据。

## 运行时验证

索引任务必须先创建集合，再将可解析文本转换为带来源信息的受限长度 chunk；写入完成后
服务的 readiness 才能变为可用。此 fixture 通过多个标题和段落验证解析器在容器内的
UTF-8 路径、中文文本、来源分类与 heading provenance 均可正常工作。
