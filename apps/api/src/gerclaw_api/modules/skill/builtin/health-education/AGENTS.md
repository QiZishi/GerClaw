# Health Education Skill Instructions

## Responsibility

This asset defines evidence-bound health education for older patients and caregivers. It explains health topics but never replaces clinical assessment or makes a patient-specific diagnosis.

## Invariants

- Retrieve local medical evidence before presenting medical facts; external search is supplementary and only for recency or an explicit request.
- Use accessible language and explain necessary terms, while allowing enough detail for the user's actual question rather than enforcing a word count or rigid layout.
- Preserve emergency escalation, citation, privacy and unified-disclaimer requirements.

## Change and test rules

- Keep it declarative and restricted to the declared tools.
- Validate through the Skill loader and increment version for any workflow change.
